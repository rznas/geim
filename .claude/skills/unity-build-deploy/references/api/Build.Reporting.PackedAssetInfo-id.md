### Description

Local file id of the object

For Serialized Files this is the local file id of the object in the file in the build output. It will be unique within that file. During the builds ids are reassigned, so this id will be different from the id of the object in its source scene or asset.

This ID is 0 for the content of .resS and .resource files.
