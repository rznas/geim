### Description

Contains information about which scenes in a build have references to an Asset in the build.

### Properties

| Property | Description |
| --- | --- |
| assetPath | The asset path. |
| scenePaths | The list of scenes in the build referring to the asset, identified by a string containing the scene index in the BuildPlayerOptions.scenes list, as well as the scene path. |
