# Vivox voice and text chat Building Block

The Vivox Building Block provides a UI-based quickstart for integrating voice and text chat into your Unity project.

| **Topic** | **Description** |
| --- | --- |
| Vivox Building Blocks prerequisites | Learn how to set up your project to work with the [Unity Services](https://docs.unity.com/en-us/services) that the Vivox Building Block uses. |
| Vivox Building Block | Add voice chat, text chat, device management, and a participant roster, to your **scene**. |
