### Description

Amount of video memory present (Read Only).

This is the approximate amount of graphics memory in megabytes.

**Note:** On WebGL platform, this property returns a fallback value of `512MB` because web browsers do not allow access to the actual GPU memory size for security and privacy reasons. This value should be treated as a safe minimal approximation rather than the actual available VRAM.

Additional resources: SystemInfo.systemMemorySize.
