### Description

Mirrors the Android property value `KEYBOARD_NOKEYS`.

For information about the property value, refer to the Android developer documentation on [KEYBOARD_NOKEYS](https://developer.android.com/reference/android/content/res/Configuration#KEYBOARD_NOKEYS).
