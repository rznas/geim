### Description

Mirrors the Android property value `UI_MODE_NIGHT_NO`.

For information about the property value, refer to the Android developer documentation on [UI_MODE_NIGHT_NO](https://developer.android.com/reference/android/content/res/Configuration#UI_MODE_NIGHT_NO).
