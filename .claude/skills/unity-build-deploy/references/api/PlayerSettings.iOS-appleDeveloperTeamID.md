### Description

Set this property with your Apple Developer Team ID. You can find this on the Apple Developer website under [Account > Membership](https://developer.apple.com/account/#/membership/) . This sets the Team ID for the generated Xcode project, allowing developers to use the Build and Run functionality. An Apple Developer Team ID must be set here for automatic signing of your app.
