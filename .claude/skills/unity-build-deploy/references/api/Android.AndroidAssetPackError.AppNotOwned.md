### Description

Indicates that the end user does not own the application on the device.

The device considers the end user to own the application only if they acquired the application from the Google Play Store. Additional resources: AndroidAssetPacks.DownloadAssetPackAsync, AndroidAssetPacks.GetAssetPackStateAsync.
