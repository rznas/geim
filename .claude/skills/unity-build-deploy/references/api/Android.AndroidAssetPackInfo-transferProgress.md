### Description

The transfering progress of the downloaded Android asset pack.

A floating point value. The range is 0 through 1. Read-only. Asset packs get compressed to reduce the download size. After they are downloaded, they have to be uncompressed. Additional resources: AndroidAssetPacks.DownloadAssetPackAsync.
