### Description

Mirrors the Android property value `NAVIGATION_UNDEFINED`.

For information about the property value, refer to the Android developer documentation on [NAVIGATION_UNDEFINED](https://developer.android.com/reference/android/content/res/Configuration#NAVIGATION_UNDEFINED).
