### Description

Mirrors the Android property value `COLOR_MODE_WIDE_COLOR_GAMUT_YES`.

For information about the property value, refer to the Android developer documentation on [COLOR_MODE_WIDE_COLOR_GAMUT_YES](https://developer.android.com/reference/android/content/res/Configuration#COLOR_MODE_WIDE_COLOR_GAMUT_YES).
