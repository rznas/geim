### Description

Options to indicate whether the screen shape is rounded or not.

Additional resources: AndroidConfiguration.screenLayoutRound

### Properties

| Property | Description |
| --- | --- |
| Undefined | Mirrors the Android property value SCREENLAYOUT_ROUND_UNDEFINED. |
| No | Mirrors the Android property value SCREENLAYOUT_ROUND_NO. |
| Yes | Mirrors the Android property value SCREENLAYOUT_ROUND_YES. |
