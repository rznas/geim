### Description

Contains information about a single step in the build process.

Additional resources: BuildReport.steps

### Properties

| Property | Description |
| --- | --- |
| depth | The nesting depth of the build step. |
| duration | The total duration for this build step. |
| messages | All log messages recorded during this build step, in the order of which they occurred. |
| name | The name of this build step. |
