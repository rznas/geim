### Description

Mirrors the Android property value `COLOR_MODE_HDR_UNDEFINED`

For information about the the property value, refer to the Android developer documentation on [COLOR_MODE_HDR_UNDEFINED](https://developer.android.com/reference/android/content/res/Configuration#COLOR_MODE_HDR_UNDEFINED).
