### Description

Mirrors the Android property value `HARDKEYBOARDHIDDEN_UNDEFINED`.

For information about the property value, refer to the Android developer documentation on [HARDKEYBOARDHIDDEN_UNDEFINED](https://developer.android.com/reference/android/content/res/Configuration#HARDKEYBOARDHIDDEN_UNDEFINED).
