### Description

XRInputSubsystem tracks all InputDevices in reference to the first known location of a specific InputDevice when set to TrackingOriginModeFlags.Device.

For mobile AR applications, the InputDevice is the mobile device the application is running on. For head-mounted device applications, the head-mounted device acts as the root.
