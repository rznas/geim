### Description

Mirrors the Android property value `TOUCHSCREEN_FINGER`.

For information about the property value, refer to the Android developer documentation on [TOUCHSCREEN_FINGER](https://developer.android.com/reference/android/content/res/Configuration#TOUCHSCREEN_FINGER).
