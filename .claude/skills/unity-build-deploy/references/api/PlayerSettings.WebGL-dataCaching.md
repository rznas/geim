### Description

Enables automatic caching of unityweb files.

When set to true, this automatically caches your content unityweb files on the users machine so they will not have to be re-downloaded on subsequent runs. Caching is implemented using the IndexedDB API provided by the browser. Some browsers may implement restrictions around this, such as asking the user for permission to cache data over a specific size.
