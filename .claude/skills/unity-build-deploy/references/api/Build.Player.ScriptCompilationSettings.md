### Description

Struct containing information on how to build scripts.

Note: this struct and its members exist to provide low-level support for the **Scriptable Build Pipeline** package. This is intended for internal use only; use the Scriptable Build Pipeline package to implement a fully featured build pipeline. You can install this via the Unity Package Manager.

### Properties

| Property | Description |
| --- | --- |
| extraScriptingDefines | User-specified preprocessor defines used while compiling assemblies. |
| group | Platform group for which scripts will be compiled. |
| options | Specific compiler options to use when compiling scripts. |
| subtarget | Subtarget for which scripts will be compiled. |
| target | Platform group for which scripts will be compiled. |
