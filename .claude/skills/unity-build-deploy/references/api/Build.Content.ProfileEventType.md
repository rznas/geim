### Description

Options for labelling captured profile events using the ContentBuildInterface.BeginProfileCapture and ContentBuildInterface.EndProfileCapture APIs.

### Properties

| Property | Description |
| --- | --- |
| Begin | Use to indicate that a task has begun. |
| End | Use to indicate that a task has ended. |
| Info | Use to indicate that general information is being reported. |
