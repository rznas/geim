### Description

Enable this setting if you want Unity to copy the PDB files containing debugging symbols to the build folder during the build process.

The PDB files provide extra debugging information about the player and other Unity libraries while debugging your build. Without this option enabled, only debug information for your own scripts is available. This option is disabled by default, and enabling it increases the size of your build.
