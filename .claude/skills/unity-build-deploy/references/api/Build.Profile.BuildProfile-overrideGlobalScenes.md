### Description

Overrides the global scene list.

When the value is true, the scenes specified in the build profile are used for building. Otherwise, the global scenes are used. Additional resources: BuildProfile.scenes, EditorBuildSettings.globalScenes, BuildProfile.GetScenesForBuild.
