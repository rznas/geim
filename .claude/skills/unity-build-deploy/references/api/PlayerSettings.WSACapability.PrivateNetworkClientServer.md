### Description

Allows your application to send and receive data on the local area network.

This capability does not allow your application to access the internet. To allow your application to receive data from the internet, set the internetClient capability. To allow your application to send or receive data from the internet, set the internetClientServer capability. This capability gives the application access to sensitive data.
