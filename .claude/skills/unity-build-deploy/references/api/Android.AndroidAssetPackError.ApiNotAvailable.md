### Description

Indicates that the Asset Delivery API is not available.

One possible cause of this is that the PlayCore version you added to the project is too old. Additional resources: AndroidAssetPacks.DownloadAssetPackAsync, AndroidAssetPacks.GetAssetPackStateAsync.
