### Description

Mirrors the Android property value `KEYBOARD_12KEY`.

For information about the property value, refer to the Android developer documentation on [KEYBOARD_12KEY](https://developer.android.com/reference/android/content/res/Configuration#KEYBOARD_12KEY).
