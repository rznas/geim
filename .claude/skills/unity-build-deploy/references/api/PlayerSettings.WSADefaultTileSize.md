### Description

Options for the application's default tile size.

For information on application tiles, see [Microsoft's documentation](https://docs.microsoft.com/en-us/windows/uwp/design/shell/tiles-and-notifications/creating-tiles#configure-the-default-tile).

### Properties

| Property | Description |
| --- | --- |
| NotSet | Doesn't use a default application tile size. |
| Medium | Uses the medium-sized tile as the default application tile size. |
| Wide | Uses the wide-sized tile as the default application tile size. |
