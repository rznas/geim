### Description

Exposes static methods for directory operations.

This is an alternative to System.IO.Directory class. Note: this class is only available when targeting Universal Windows Platform.

### Static Properties

| Property | Description |
| --- | --- |
| localFolder | Returns a path to local folder. |
| roamingFolder | Returns a path to roaming folder. |
| temporaryFolder | Returns a path to temporary folder. |

### Static Methods

| Method | Description |
| --- | --- |
| CreateDirectory | Creates directory in the specified path. |
| Delete | Deletes a directory from a specified path. |
| Exists | Determines whether the given path refers to an existing directory. |
