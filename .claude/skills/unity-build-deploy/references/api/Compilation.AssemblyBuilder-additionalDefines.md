### Description

Additional #define directives passed to compilation of the assembly.

By default the same #define directives that are added to scripts inside the Assets folder are added the assembly build. This property allows to add additional #define directives.

The default #define directives are in part controlled by AssemblyBuilder.flags, AssemblyBuilder.buildTarget and AssemblyBuilder.buildTargetGroup.

Additional resources: AssemblyBuilder.defaultDefines.
