### Description

Contains information about a single file produced by the build process.

### Properties

| Property | Description |
| --- | --- |
| id | The unique indentifier of the build file. |
| path | The absolute path of the file produced by the build process. |
| role | The role the file plays in the build output. |
| size | The total size of the file, in bytes. |
