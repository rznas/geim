### Description

Specifies the available options for the Unity engine code to use in non-development builds. Only supported on Meta Quest.

You can set the value of this enum to linkTimeOptimization property. For more information, refer to Link Time Optimization.

### Properties

| Property | Description |
| --- | --- |
| None | The project build uses the default Unity engine code. |
| Thin | The project build uses precompiled and thin link time optimized (ThinLTO) Unity engine code. |
