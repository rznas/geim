### Description

Represents a container that you can use to specify additional dependencies for the AndroidProjectFilesModifier process depends.

### Properties

| Property | Description |
| --- | --- |
| DependencyFiles | An array of dependency files paths. |
| DependencyFilesAsGUIDs | An array of dependency files as asset GUIDs. |
