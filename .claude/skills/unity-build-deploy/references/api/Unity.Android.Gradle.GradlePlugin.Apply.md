### Description

The C# definition of the `apply` property.

The `apply` statement with a boolean can be used to disable the default behavior of applying the plugin immediately (e.g. you want to apply it only in `subprojects`). For more information about the element, see Gradle's documentation: Applying plugins with the plugins DSL
