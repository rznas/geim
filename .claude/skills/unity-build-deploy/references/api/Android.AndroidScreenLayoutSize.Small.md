### Description

Mirrors the Android property value `SCREENLAYOUT_SIZE_SMALL`.

For information about the property value, refer to the Android developer documentation on [SCREENLAYOUT_SIZE_SMALL](https://developer.android.com/reference/android/content/res/Configuration#SCREENLAYOUT_SIZE_SMALL).
