### Description

Runs the app on a remote Windows device using the Windows DevicePortal (WDP) transport.

To use this option, DevicePortal must be enabled on the target device and the connection address must be provided. Additionally, if DevicePortal authentication is enabled, the WDP username and password must also be provided.
