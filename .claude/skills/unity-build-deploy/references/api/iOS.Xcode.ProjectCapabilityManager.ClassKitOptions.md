### Description

The list of options available for the ClassKit capability. For more information, refer to [ClassKit Environment Entitlement (Apple Developer)](https://developer.apple.com/documentation/bundleresources/entitlements/com_apple_developer_classkit-environment).

### Properties

| Property | Description |
| --- | --- |
| Development | Use this value to enable the Development option. |
| Production | Use this value to enable the Production option. |
