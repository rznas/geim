### Description

Includes all App icons required by tvOS.

Small icon (400x240) displayed on the home screen
 Small icon 2x (800x480) displayed on the 4K Apple TV home screen.
 Large icon (1280x720) displayed on the app's App Store page
 All icons require at least two but no more than five layers.
