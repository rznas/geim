### Description

Allows your application to interact with connected Human Interface Devices (HIDs).

This is necessary if you want to use input devices or controllers that Windows does not natively support. To access devices, you must set additional device capabilities within the package manifest. For information on HID capabilities, see [Microsoft's documentation](https://docs.microsoft.com/en-us/uwp/schemas/appxpackage/how-to-specify-device-capabilities-for-hid).
