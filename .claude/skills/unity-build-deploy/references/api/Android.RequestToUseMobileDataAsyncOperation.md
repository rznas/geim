### Description

Represents an asynchronous operation that requests to use mobile data to download Android asset packs.

AndroidAssetPacks.RequestToUseMobileDataAsync returns an instance of this class. You can yield until the operation completes, or manually check whether it's done using isDone or keepWaiting properties. Additional resources: AndroidAssetPacks.RequestToUseMobileDataAsync.

### Properties

| Property | Description |
| --- | --- |
| isDone | Checks if the operation is finished. |
| keepWaiting | Checks if the operation is still running. |
| result | Indicates whether the end user allowed the application to use mobile data to download Android asset packs. |
