### Description

Options for filtering captured profile events using the ContentBuildInterface.BeginProfileCapture and ContentBuildInterface.EndProfileCapture APIs.

### Properties

| Property | Description |
| --- | --- |
| None | Use to exclude none of the captured events. |
| IgnoreShortEvents | Use to exclude all captured events that are less than 10 microseconds in duration. |
