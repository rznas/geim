### Description

The list of options available for the 5G Network Slicing app category key. For more information, refer to [5G Network Slicing App Category (Apple Developer)](https://developer.apple.com/documentation/bundleresources/entitlements/com_apple_developer_networking_slicing_appcategory)

### Properties

| Property | Description |
| --- | --- |
| None | Use this flag when additional 5G Network Slicing app category options aren't necessary. |
| Gaming | Use this flag to set Gaming option. |
| Communication | Use this flag to set the Communication option. |
