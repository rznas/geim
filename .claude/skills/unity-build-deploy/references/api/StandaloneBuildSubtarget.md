### Description

Options for the desktop platform subtarget type.

Additional resources: EditorUserBuildSettings.standaloneBuildSubtarget.

### Properties

| Property | Description |
| --- | --- |
| Player | Uses the default desktop player build settings. |
| Server | Uses a player build optimized to run as a headless server. |
