### Description

Unity does not include native debug metadata in the application build.

As a result, your application doesn't contain any debugging information. Select this enum value when you don't require a symbol package, such as when native debugging isn't needed or when you don't need to upload symbol files to the Play Store. This option results in slightly faster builds.
