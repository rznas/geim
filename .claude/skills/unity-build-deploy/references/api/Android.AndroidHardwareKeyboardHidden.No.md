### Description

Mirrors the Android property value `HARDKEYBOARDHIDDEN_NO`.

For information about the property value, refer to the Android developer documentation on [HARDKEYBOARDHIDDEN_NO](https://developer.android.com/reference/android/content/res/Configuration#HARDKEYBOARDHIDDEN_NO).
