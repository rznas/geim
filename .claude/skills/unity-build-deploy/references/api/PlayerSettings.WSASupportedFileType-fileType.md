### Description

The file extension that identifies the file type.

The extension must follow the following Windows filename requirements:

- Between one and 64 characters long.
- Must begin with a period (".").
- Cannot have additional periods.
- Cannot contain the following characters: <, >, :, ", /, \, |, ?, or *.

For example, ".jpeg".
