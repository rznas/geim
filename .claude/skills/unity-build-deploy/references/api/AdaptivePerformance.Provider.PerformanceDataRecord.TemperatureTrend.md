### Description

The current temperature trend in the range of [-1.0, 1.0] that is a metric of temperature change over time. The temperature trend is constant at 0.0 in case the feature is not supported. The temperature trend has changed when Feature.TemperatureTrend bit is set in ChangeFlags.
