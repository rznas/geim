### Description

Vertex tangents

Used by XRMeshSubsystem.GenerateMeshAsync to request tangents be generated, or by MeshGenerationResult to indicate tangents were written to the Mesh.

 Additional resources: XRMeshSubsystem.GenerateMeshAsync, MeshGenerationResult
