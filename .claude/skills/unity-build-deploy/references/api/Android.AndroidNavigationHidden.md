### Description

Options to indicate whether the 5-way or DPAD navigation methods are available on the device.

Additional resources: AndroidConfiguration.navigationHidden

### Properties

| Property | Description |
| --- | --- |
| Undefined | Mirrors the Android property value NAVIGATIONHIDDEN_UNDEFINED. |
| No | Mirrors the Android property value NAVIGATIONHIDDEN_NO. |
| Yes | Mirrors the Android property value NAVIGATIONHIDDEN_YES. |
