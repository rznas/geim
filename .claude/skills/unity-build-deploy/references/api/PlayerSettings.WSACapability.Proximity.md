### Description

Allows your application to initiate connections between multiple devices in close proximity to communicate with one another.

This capability is useful in casual multiplayer games and in apples that exchange information. Devices attempt to use communication technology that provinces the best possible connection. This includes Bluetooth, Wi-Fi, and the internet.
