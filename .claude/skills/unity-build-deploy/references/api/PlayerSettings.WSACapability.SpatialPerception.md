### Description

(Deprecated) Allows your application to access spatial mapping data through a mixed reality device.

This capability provides mixed reality applications with information about surfaces in application-specified regions of space near the user. Applications that perform holographic rendering based on the end user's head post do not require this capability. Only set this capability when your application needs access to these surface meshes.
