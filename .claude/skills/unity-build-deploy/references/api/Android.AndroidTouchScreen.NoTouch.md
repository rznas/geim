### Description

Mirrors the Android property value `TOUCHSCREEN_NOTOUCH`.

For information about the property value, refer to the Android developer documentation on [TOUCHSCREEN_NOTOUCH](https://developer.android.com/reference/android/content/res/Configuration#TOUCHSCREEN_NOTOUCH).
