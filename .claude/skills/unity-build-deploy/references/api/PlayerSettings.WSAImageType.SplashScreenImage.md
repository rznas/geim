### Description

The image to display as the splash screen while the Universal Windows Platform application opens.

The Universal Windows Platform asset filename for this is SplashScreen.png. For more information about application icons and logos, see [Microsoft's documentation](https://docs.microsoft.com/en-us/windows/uwp/design/style/app-icons-and-logos).
