### Description

The list of options available for the Fonts capability. For more information, refer to [Configuring custom fonts (Apple Developer)](https://developer.apple.com/documentation/xcode/configuring-custom-fonts).

### Properties

| Property | Description |
| --- | --- |
| None | Use this flag when custom fonts aren't necessary. |
| InstallFonts | Use this flag to set the Install Fonts option. |
| UseInstalledFonts | Use this flag to set the Use Installed Fonts option. |
