### Description

Converts a managed array of System.Byte to a Java array of `byte`.

AndroidJNI.ToByteArray is obsolete. Use AndroidJNI.ToSByteArray instead.

Additional resources: AndroidJNI.NewByteArray, AndroidJNI.SetByteArrayElement.
