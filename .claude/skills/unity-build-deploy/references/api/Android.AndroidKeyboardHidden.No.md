### Description

Mirrors the Android property value `KEYBOARDHIDDEN_NO`.

For information about the property value, refer to the Android developer documentation on [KEYBOARDHIDDEN_NO](https://developer.android.com/reference/android/content/res/Configuration#KEYBOARDHIDDEN_NO).
