### Description

Code optimization mode defines whether UnityEditor compiles scripts in Debug or Release mode.

### Properties

| Property | Description |
| --- | --- |
| None | Unitialized value for code optimization mode. |
| Debug | Debug mode enables C# debugging but reduces C# performance. |
| Release | Release mode disables C# debugging but improves C# performance. |
