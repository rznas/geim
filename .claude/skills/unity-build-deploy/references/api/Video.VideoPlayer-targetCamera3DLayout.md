### Description

Type of 3D content contained in the source video media.

When stereoscopic rendering is used, the rendering for each eye samples the correct half of the video according to this setting. This setting is only used when VideoPlayer.renderMode is set to either Video.VideoTarget.CameraBackPlane or Video.VideoTarget.CameraFrontPlane.
