### Description

Single layer icons primarily used by devices running Android 7.0 (API level 24) or below.

**Note**: `AndroidPlatformIconKind.Legacy` is deprecated and will be removed in a future version of Unity. Use AndroidPlatformIconKind.Adaptive instead.
