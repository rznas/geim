### Description

Mirrors the Android property value `SCREENLAYOUT_LAYOUTDIR_LTR`.

For information about the property value, refer to the Android developer documentation on [SCREENLAYOUT_LAYOUTDIR_LTR](https://developer.android.com/reference/android/content/res/Configuration#SCREENLAYOUT_LAYOUTDIR_LTR).
