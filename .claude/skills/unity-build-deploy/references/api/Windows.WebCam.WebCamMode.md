### Description

Describes the active mode of the Web Camera resource.

The Web Camera resource can only be in one mode at any given time.

### Properties

| Property | Description |
| --- | --- |
| None | Resource is not in use. |
| PhotoMode | Resource is in Photo Mode. |
| VideoMode | Resource is in Video Mode. |
