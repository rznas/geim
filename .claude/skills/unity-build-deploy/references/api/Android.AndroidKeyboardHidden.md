### Description

Options to indicate whether any keyboard is available for use on the device.

Additional resources: AndroidConfiguration.keyboardHidden

### Properties

| Property | Description |
| --- | --- |
| Undefined | Mirrors the Android property value KEYBOARDHIDDEN_UNDEFINED. |
| No | Mirrors the Android property value KEYBOARDHIDDEN_NO. |
| Yes | Mirrors the Android property value KEYBOARDHIDDEN_YES. |
