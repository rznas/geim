### Description

Represents the file types to register the application as a default handler for.

To set file type associations, use the fileTypeAssociations Property. 
For information on Windows file type association, see [Microsoft's documentation](https://docs.microsoft.com/en-us/windows/win32/shell/fa-intro). 
Additional resources: WSASupportedFileType.

### Properties

| Property | Description |
| --- | --- |
| name | The name of the file type association. This is to organize and group file types. |
| supportedFileTypes | The set of file and content types your application supports. |
