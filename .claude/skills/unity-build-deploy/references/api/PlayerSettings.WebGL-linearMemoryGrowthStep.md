### Description

Heap memory growth step in MB.

Advanced tuning option to control the growth step for the heap memory in MB if linear memory growth is enabled (Recommended default: 16 MB). A growth step of 16 MB means the size of the heap is increased by 16 MB each time the heap needs to grow.
