### Description

Introduced in Android 7.1 (API level 25).

**Note**: `AndroidPlatformIconKind.Round` is deprecated and will be removed in a future version of Unity. Use AndroidPlatformIconKind.Adaptive instead. To display a round icon for your application on devices running API 26 and later, use AndroidPlatformIconKind.Adaptive.
