### Description

Build configurations for Windows Store Visual Studio solutions.

### Properties

| Property | Description |
| --- | --- |
| Debug | Debug configuation. No optimizations, profiler code enabled. |
| Release | Release configuration. Optimization enabled, profiler code enabled. |
| Master | Master configuation. Optimizations enabled, profiler code disabled. This configuration is used when submitting the application to Windows Store. |
