### Description

Mirrors the Android property value `ORIENTATION_PORTRAIT`.

For information about the property value, refer to the Android developer documentation on [ORIENTATION_PORTRAIT](https://developer.android.com/reference/android/content/res/Configuration#ORIENTATION_PORTRAIT).
