### Returns

**string** The default main target name.

### Description

Deprecated. Returns the default main target name in Unity project.

Deprecated. The returned target name can then be used to retrieve the GUID of the target via TargetGuidByName function. This function can only be used in Unity-generated projects. Use GetUnityMainTargetGuid or GetUnityFrameworkTargetGuid to get GUID. You need not call TargetGuidByName function.
