### Description

Indicates an error which the device encountered when downloading the Android asset pack.

The default value is AndroidAssetPackError.NoError which indicates no errors. Read-only. Additional resources: AndroidAssetPackError, AndroidAssetPacks.DownloadAssetPackAsync.
