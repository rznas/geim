### Description

Represents the current status of the speech recognition system or a dictation recognizer.

### Properties

| Property | Description |
| --- | --- |
| Stopped | Speech recognition system is stopped. |
| Running | Speech recognition system is running. |
| Failed | Speech recognition system has encountered an error and is in an indeterminate state. |
