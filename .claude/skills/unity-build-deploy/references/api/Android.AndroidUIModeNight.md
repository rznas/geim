### Description

Options to indicate whether the device screen is in a special mode, such as a night mode.

Additional resources: AndroidConfiguration.uiModeNight

### Properties

| Property | Description |
| --- | --- |
| Undefined | Mirrors the Android property value UI_MODE_NIGHT_UNDEFINED. |
| No | Mirrors the Android property value UI_MODE_NIGHT_NO. |
| Yes | Mirrors the Android property value UI_MODE_NIGHT_YES. |
