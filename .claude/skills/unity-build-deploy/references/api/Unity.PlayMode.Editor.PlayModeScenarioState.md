### Description

Represents the different states of a play mode scenario.

### Properties

| Property | Description |
| --- | --- |
| Idle | The current play mode scenario is not running. |
| Starting | The current play mode scenario is starting. |
| Running | The current play mode scenario is running. |
| Stopping | The current play mode scenario is stopping. |
