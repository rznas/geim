### Description

Draw the video content into a user-specified property of the current GameObject's material.

This mode is simpler to use than VideoRenderMode.RenderTexture. Like RenderTexture, use to show a video on the texture of scene objects like TVs, computers, etc. However, you can’t show the same video on multiple objects unless you duplicate the materials or VideoPlayer.
