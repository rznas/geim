### Description

Converts a Java array of `byte` to a managed array of System.Byte.

AndroidJNI.FromByteArray is obsolete. Use AndroidJNI.FromSByteArray instead.

Additional resources: AndroidJNI.GetByteArrayElement, AndroidJNI.GetArrayLength.
