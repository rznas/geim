### Description

Mirrors the Android property value `NAVIGATION_NONAV`.

For information about the property value, refer to the Android developer documentation on [NAVIGATION_NONAV](https://developer.android.com/reference/android/content/res/Configuration#NAVIGATION_NONAV).
