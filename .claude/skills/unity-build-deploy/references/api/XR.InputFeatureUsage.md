### Description

Defines a generic usage that maps to an input feature on a device. Use the As method to turn into a generic usage.

### Properties

| Property | Description |
| --- | --- |
| name | The string name of this usage feature; used internally to map to an input feature on a device. |
| type | The type of this usage feature; used internally to map to an input feature on a device. |

### Public Methods

| Method | Description |
| --- | --- |
| As | Returns the generic version of this type for retrieving a feature value from a device. |
