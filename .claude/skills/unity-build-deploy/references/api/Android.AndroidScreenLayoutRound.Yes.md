### Description

Mirrors the Android property value `SCREENLAYOUT_ROUND_YES`.

For information about the property value, refer to the Android developer documentation on [SCREENLAYOUT_ROUND_YES](https://developer.android.com/reference/android/content/res/Configuration#SCREENLAYOUT_ROUND_YES).
