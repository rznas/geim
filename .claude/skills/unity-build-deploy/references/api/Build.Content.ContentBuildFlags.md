### Description

Build options for content.

Note: this enum and its values exist to provide low-level support for the **Scriptable Build Pipeline** package. This is intended for internal use only; use the Scriptable Build Pipeline package to implement a fully featured build pipeline. You can install this via the Unity Package Manager.

### Properties

| Property | Description |
| --- | --- |
| None | Build content with no additional options. |
| DisableWriteTypeTree | Do not include type information within the built content. |
| StripUnityVersion | Build Flag to indicate the Unity Version should not be written to the serialized file. |
| DevelopmentBuild | Build a development version of the content files. |
