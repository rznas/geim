### Description

iOS launch screen settings.

### Properties

| Property | Description |
| --- | --- |
| Default | Use the default launch screen (dark blue background). |
| ImageAndBackgroundRelative | Use a custom launch screen image specified in the iOS Player Settings or with PlayerSettings.iOS.SetLaunchScreenImage which will be scaled across the entire screen. |
| ImageAndBackgroundConstant | Use a custom launch screen image specified in the iOS Player Settings or with PlayerSettings.iOS.SetLaunchScreenImage and use its original dimensions. |
| CustomStoryboard | Use a specified custom Storyboard (.storyboard) file in Player Settings. |
