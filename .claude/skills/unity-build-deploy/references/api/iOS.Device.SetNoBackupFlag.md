### Description

Set file flag to be excluded from iCloud/iTunes backup. This uses [NSURLIsExcludedFromBackupKey](https://developer.apple.com/documentation/foundation/nsurlisexcludedfrombackupkey). Note that you should set this property each time you save a file.
