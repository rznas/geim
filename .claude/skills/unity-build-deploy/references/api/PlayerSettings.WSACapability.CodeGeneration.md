### Description

Allows your application to access specific APIs that provide just-in-time compilation.

This capability gives you access to:

 * [VirtualProtectFromApp](https://docs.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-virtualprotectfromapp?redirectedfrom=MSDN) * [CreateFileMappingFromApp](https://docs.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-createfilemappingfromapp?redirectedfrom=MSDN) * [OpenFileMappingFromApp](https://docs.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-openfilemappingfromapp?redirectedfrom=MSDN) * [MapViewOfFileFromApp](https://docs.microsoft.com/en-us/windows/win32/api/memoryapi/nf-memoryapi-mapviewoffilefromapp?redirectedfrom=MSDN)
