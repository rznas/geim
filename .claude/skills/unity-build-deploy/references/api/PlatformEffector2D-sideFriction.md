### Description

Whether friction should be used on the platform sides or not.

This is useful to stop friction slowing a Collider2D when in contact with a vertical surface when a force is being applied to keep the Collider2D in contact with the surface. The "sides" are defined as the edges perpendicular to the "top" surface(s) in local-space.
