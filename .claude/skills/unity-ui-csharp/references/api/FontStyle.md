### Description

Font Style applied to GUI Texts, Text Meshes or GUIStyles.

### Properties

| Property | Description |
| --- | --- |
| Normal | No special style is applied. |
| Bold | Bold style applied to your texts. |
| Italic | Italic style applied to your texts. |
| BoldAndItalic | Bold and Italic styles applied to your texts. |
