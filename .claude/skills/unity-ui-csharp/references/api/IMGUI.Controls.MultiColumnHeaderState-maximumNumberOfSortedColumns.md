### Description

This property controls the maximum number of columns returned by the sortedColumns property.

This property controls the maximum number of columns returned by the sortedColumns property. It is set to a default value of 3 which means that the 3 most recent changes to column sorting are stored. Use the sortedColumns property to sort your data by multiple columns.
