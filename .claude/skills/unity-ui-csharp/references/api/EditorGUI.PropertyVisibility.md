### Description

Specifies which SerializedProperties are drawn by EditorGUI.MultiPropertyField.

### Properties

| Property | Description |
| --- | --- |
| All | Draw all SerializedProperties, even if it is not visible. |
| OnlyVisible | Draw only visible SerializedProperties. |
