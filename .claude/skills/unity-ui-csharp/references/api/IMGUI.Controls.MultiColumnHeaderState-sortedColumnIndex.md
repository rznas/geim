### Description

This property holds the index to the primary sorted column.

This property holds the index to the primary sorted column with the default value of -1 (no sorting).

Use the sortedColumns array which contains the history of the most recent sorted columns, to sort multiple columns. The first entry in that array is the primary sorted column.
