### Description

A flag enumeration for specifying which axes on a PrimitiveBoundsHandle object should be enabled.

### Properties

| Property | Description |
| --- | --- |
| None | No axes. |
| X | X-axis (bit 0). |
| Y | Y-axis (bit 1). |
| Z | Z-axis (bit 2). |
| All | All axes. |
