### Description

Determines how toolbar button size is calculated.

### Properties

| Property | Description |
| --- | --- |
| Fixed | Calculates the button size by dividing the available width by the number of buttons. The minimum size is the maximum content width. |
| FitToContents | The width of each toolbar button is calculated based on the width of its content. |
