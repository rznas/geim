### Description

Interface for UXML upgrade operations that can be applied to VisualTreeAsset. Inherit from this interface to create a new UXML upgrader.

### Properties

| Property | Description |
| --- | --- |
| description | Description of what this upgrader does. |
| name | The name of this upgrader, displayed to users. |

### Public Methods

| Method | Description |
| --- | --- |
| Upgrade | Apply the upgrade to the asset. |
