### Description

Contains timing information of IVisualElementScheduler events.

### Properties

| Property | Description |
| --- | --- |
| deltaTime | Time difference in milliseconds between now and the previous callback. |
| now | Current time in milliseconds. |
| start | Start time in milliseconds, or last callback time for repeatable IScheduledItem. |
