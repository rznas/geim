### Description

Defines how an element is displayed in the layout.

Unlike the visibility property, this property affects the layout of the element. This is a convenient way to hide an element without removing it from the hierarchy (when using the DisplayStyle.None). Elements with a display style of DisplayStyle.None are ignored by pointer events and by IPanel.Pick.
