### Description

Options to change the drag-and-drop mode for items in the ListView.

Using `Animated` will affect the layout of the ListView, by adding drag handles before every item. Multiple item drag is only supported in the `Simple` mode.

### Properties

| Property | Description |
| --- | --- |
| Simple | ListView will display the standard blue line dragger on reorder. |
| Animated | ListView will add drag handles before every item, that can be used to drag a single item with animated visual feedback. |
