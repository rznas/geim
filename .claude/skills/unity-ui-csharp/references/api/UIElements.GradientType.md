### Description

Specifies the type of gradient to use for color interpolation.

Linear creates a gradient that transitions colors along a straight line. Radial creates a gradient that radiates colors outward from a central point.

### Properties

| Property | Description |
| --- | --- |
| Linear | A gradient that transitions colors along a straight line. |
| Radial | A gradient that radiates colors outward from a central point. |
