### Description

Status of the menu item.

Use the values of this enumeration as flags

### Properties

| Property | Description |
| --- | --- |
| None | Do not display the item. |
| Normal | Display the item normally. |
| Disabled | Disable the item and make it so it is not selectable by the user. |
| Checked | Display the item with a checkmark. |
| Hidden | Do not display the item. |
