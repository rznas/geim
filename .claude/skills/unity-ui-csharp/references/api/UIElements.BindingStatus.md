### Description

Status to report the result of a binding update.

### Properties

| Property | Description |
| --- | --- |
| Success | Indicates that the binding has successfully processed the update. |
| Failure | Indicates that the binding has failed to process the update. |
| Pending | Indicates that the binding has not yet completed the update. |
