### Description

Use this enum to specify the default dock position of the element: the left, the middle, or the right.

Additional resources: MainToolbarElementAttribute.

### Properties

| Property | Description |
| --- | --- |
| Left | Specifies the left dock of the main toolbar. |
| Right | Specifies the right dock of the main toolbar. |
| Middle | Specifies the middle dock of the main toolbar. |
