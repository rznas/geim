### Description

The CapFunction to use when displaying the angle control handle.

Defaults to ArcHandle.DefaultAngleHandleDrawFunction for new instances of JointAngularLimitHandle. No angle handle will be drawn if a value of `null` is assigned.

Additional resources: angleHandleSizeFunction.
