### Parameters

| Parameter | Description |
| --- | --- |
| rect | Rect to render the text in. |
| label | Label to render. |
| selected | Selected state used for determining text color. |
| focused | Focused state used for determining text color. |

### Description

Draws a bold label that have correct text color when selected and/or focused.

This method uses the normal, focused, onNormal and onFocused GUIStyleStates of TreeView.DefaultStyles.boldLabel to ensure correct rendering of text above a selection background.
