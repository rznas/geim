### Description

Configure ScrollView for vertical and horizontal scrolling.

Requires elements with the height property explicitly defined. A ScrollView configured with this mode has the ScrollView.verticalHorizontalVariantUssClassName class in its class list. The difference with the vertical mode is that content will not wrap.
