### Description

The array of column indices for multiple column sorting.

Use this array to sort your data by multiple columns. The first entry in the array is the primary sorted column. The sorting order for a column index can be determined by using the MultiColumnHeader.IsSortedAscending.

Additional resources: maximumNumberOfSortedColumns, sortedColumnIndex.
