### Description

Options to specify the glyph class definition type.

### Properties

| Property | Description |
| --- | --- |
| Undefined | Defines the glyph as Undefined. |
| Base | Defines the glyph as a Base glyph. |
| Ligature | Defines the glyph as a Ligature. |
| Mark | Defines the glyph as a Mark glyph. |
| Component | Defines the glyph as a Component glyph. |
