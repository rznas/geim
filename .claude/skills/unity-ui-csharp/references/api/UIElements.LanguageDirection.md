### Description

Indicates the directionality of the element's text. The value cascades to child elements.

Setting `languageDirection` to `RTL` can only get the basic RTL support like text reversal. To get more comprehensive RTL support, such as line breaking, word wrapping, or text shaping, you must enable Advance Text Generator. 

 Additional resources: Language direction

### Properties

| Property | Description |
| --- | --- |
| Inherit | Inherits the directionality from the nearest ancestor with a specified directionality. |
| LTR | Left to right language direction. |
| RTL | Right to left language direction. |
