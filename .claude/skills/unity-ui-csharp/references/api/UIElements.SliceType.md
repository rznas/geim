### Description

The slice type determines whether the center of the image is scaled or tiled.

### Properties

| Property | Description |
| --- | --- |
| Sliced | Fill the slices by stretching the center and sides. |
| Tiled | Fill the slices by tiling the center and sides. Image must be imported as a Sprite (2D and UI) and have Mesh Type set to Full Rect. |
