### Description

The version of HTTP to force when using UnityWebRequest.

### Properties

| Property | Description |
| --- | --- |
| NotForced | Do not force any HTTP version when using UnityWebRequest. |
| HTTP1_0 | Force HTTP/1.0 when using UnityWebRequest. |
| HTTP1_1 | Force HTTP/1.1 when using UnityWebRequest. |
| HTTP2 | Force HTTP/2 when using UnityWebRequest. |
