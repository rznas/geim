### Description

Whether to override the build flags specified when creating a RayTracingAccelerationStructure.

If you set this value to `true`, Unity uses RayTracingMultiGeometryInstanceConfig.accelerationStructureBuildFlags when it builds the acceleration structure for the geometries referenced by the ray tracing instances specified using this configuration.
