### Description

An array of all the keyword declaration overrides in this settings configuration.

Setting the value of this property will run validation on the data and throw an `ArgumentException` in the case of invalid data. This property can only be used for setting the override data. To read the current data the ShaderBuildSettings.GetKeywordDeclarationOverridesCopy method must be used.
