### Description

Create a GraphicsStateCollection populated with contents of a file, or empty.

The file extension of GraphicsStateCollection is *.graphicsstate. **Note**: On the Web platform, you can’t load a GraphicsStateCollection from a file because the platform doesn’t support it. Additional resources: LoadFromFile.
