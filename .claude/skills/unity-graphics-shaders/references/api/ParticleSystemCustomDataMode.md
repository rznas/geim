### Description

Which mode CustomDataModule uses to generate its data.

### Properties

| Property | Description |
| --- | --- |
| Disabled | Don't generate any data. |
| Vector | Generate data using MinMaxCurve. |
| Color | Generate data using MinMaxGradient. |
