### Description

Whether to override the build flags specified when creating a RayTracingAccelerationStructure.

If you set this value to `true`, Unity uses RayTracingMeshInstanceConfig.accelerationStructureBuildFlags when it builds the acceleration structure for the geometry referenced by this ray tracing instance.
