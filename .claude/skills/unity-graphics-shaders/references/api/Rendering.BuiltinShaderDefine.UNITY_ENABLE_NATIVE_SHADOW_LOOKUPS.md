### Description

UNITY_ENABLE_NATIVE_SHADOW_LOOKUPS enables use of built-in shadow comparison samplers on OpenGL ES 2.0.

Deprecated: Unity no longer sets UNITY_ENABLE_NATIVE_SHADOW_LOOKUPS on any platform.
