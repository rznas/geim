### Description

Static factory class for creating machine learning operators.

### Static Methods

| Method | Description |
| --- | --- |
| Conv | Creates a convolution operator. |
| Gemm | Creates a general matrix multiplication (GEMM) operator. |
| Identity | Creates an identity operator. |
| Reduce | Creates a reduce operator. |
