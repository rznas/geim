### Description

The type of shape used for influencing particles in the Force Field Component.

### Properties

| Property | Description |
| --- | --- |
| Sphere | Influence particles inside a sphere shape. |
| Hemisphere | Influence particles inside a hemisphere shape. |
| Cylinder | Influence particles inside a cylinder shape. |
| Box | Influence particles inside a box shape. |
