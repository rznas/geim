### Description

With HideSelectedItems you can hide selected filter items in order to easily discard uninteresting Collider properties thereby making overview and navigation easier.

For example **Triggers** can easily be ignored.

You can also hide **Sleeping Rigid Bodies** to only see the Rigidbodies that are calculated in the physics simulation.
