### Description

Prewarms shaders in a way that is supported by all graphics APIs.

For information on shader loading and prewarming, see Shader loading.

### Static Methods

| Method | Description |
| --- | --- |
| WarmupShader | Prewarms all shader variants for a given Shader, using a given rendering configuration. |
| WarmupShaderFromCollection | Prewarms the shader variants for a given Shader that are in a given ShaderVariantCollection, using a given rendering configuration. |
