### Description

Whether shader keyword filter attributes include the keywords, remove the keywords or do nothing, based on the attribute condition evaluation.

### Properties

| Property | Description |
| --- | --- |
| Select | Includes only the keywords in any multi_compile keyword set containing them, if the attribute condition evaluates as true. |
| Remove | Removes the keywords from any multi_compile keyword set containing them, if the attribute condition evaluates as true. |
| SelectOrRemove | Includes only the keywords if the attribute condition evaluates as true. Removes the keywords if the attribute condition evaluates as false. |
