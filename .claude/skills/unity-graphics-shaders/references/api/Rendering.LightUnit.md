### Description

The unit of a Light's intensity.

Additional resources: Light.lightUnit, light component.

### Properties

| Property | Description |
| --- | --- |
| Lumen | The unit of luminous flux. |
| Candela | The base unit of luminous intensity in the International System of Units. |
| Lux | The unit of illuminance (lumen per square meter). |
| Nits | The unit of luminance (candela per square meter). |
| Ev100 | A value that represents a combination of a camera's shutter speed and f-number. |
