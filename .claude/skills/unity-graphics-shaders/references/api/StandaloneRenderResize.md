### Description

Enum used to determine if a Canvas should be resized when a manual Camera.Render call is performed.

### Properties

| Property | Description |
| --- | --- |
| Enabled | Resize the Canvas when a manual Camera.Render call is performed. |
| Disabled | Do not resize the Canvas when a manual Camera.Render call is performed. |
