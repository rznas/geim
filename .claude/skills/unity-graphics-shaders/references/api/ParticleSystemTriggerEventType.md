### Description

The different types of particle triggers.

### Properties

| Property | Description |
| --- | --- |
| Inside | Trigger when particles are inside the collision volume. |
| Outside | Trigger when particles are outside the collision volume. |
| Enter | Trigger when particles enter the collision volume. |
| Exit | Trigger when particles leave the collision volume. |
