### Description

Trilight ambient lighting.

Ambient is defined by three colors: "sky", "equator" and "ground".

Additional resources: RenderSettings.ambientSkyColor, RenderSettings.ambientEquatorColor, RenderSettings.ambientGroundColor, RenderSettings.ambientMode, Lighting Window.
