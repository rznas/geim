### Description

Specifies the OpenGL ES version.

Additional resources: Graphics.minOpenGLESVersion

### Properties

| Property | Description |
| --- | --- |
| None | No valid OpenGL ES version |
| OpenGLES30 | OpenGL ES 3.0 |
| OpenGLES31 | OpenGL ES 3.1 |
| OpenGLES31AEP | OpenGL ES 3.1 with Android Extension Pack |
| OpenGLES32 | OpenGL ES 3.2 |
