### Description

Utilities for Camera rendering in the Editor.

### Static Methods

| Method | Description |
| --- | --- |
| GetRenderersFilteringResults | Returns the SceneView scene filtering results for a specified NativeArray of Renderer instance IDs. |
| GetRenderersHiddenResultBits | Gets the scene visibility status of renderers. |
| RenderToCubemap | Renders this Camera into a static cubemap. |
