### Description

An enum that represents graphics tiers.

**Note:** Graphics tiers are only supported in the Built-in Render Pipeline.

Additional resources: Graphics tiers, TierSettings, Graphics.activeTier, EditorGraphicsSettings.GetTierSettings, EditorGraphicsSettings.SetTierSettings.

### Properties

| Property | Description |
| --- | --- |
| Tier1 | The lowest graphics tier. Corresponds to low-end devices. |
| Tier2 | The medium graphics tier. Corresponds to mid-range devices. |
| Tier3 | The highest graphics tier. Corresponds to high-end devices. |
