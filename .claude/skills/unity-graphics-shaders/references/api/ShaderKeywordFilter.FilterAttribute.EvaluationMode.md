### Description

How Unity evaluates the shader keyword filter attribute state.

### Properties

| Property | Description |
| --- | --- |
| Normal | The shader keyword filter is active if the condition matches the data field the attribute is attached to. |
| Negated | The shader keyword filter is active if the condition does not match the data field the attribute is attached to. |
