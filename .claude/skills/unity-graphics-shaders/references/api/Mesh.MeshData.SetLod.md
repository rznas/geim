### Parameters

| Parameter | Description |
| --- | --- |
| submesh | Submesh to set LOD for. |
| level | Level to set. |
| levelRange | Index range for the set LOD. |
| flags | Mesh LOD update flags. If set to DontValidateIndices, then index range validation is skipped. |

### Description

Set a Mesh LOD range for a given submesh and level of detail.
