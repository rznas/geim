### Parameters

| Parameter | Description |
| --- | --- |
| name | The name of the property. |
| nameID | The name ID of the property. Use Shader.PropertyToID to get this ID. |

### Returns

**bool** Returns true if MaterialPropertyBlock has this property.

### Description

Checks if MaterialPropertyBlock has the Texture property with the given name or name ID. To set the property, use SetTexture.
