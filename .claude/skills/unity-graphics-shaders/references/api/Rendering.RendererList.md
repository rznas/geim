### Description

Represents a set of visible GameObjects.

To create a RendererList, use the ScriptableRenderContext.CreateRendererList, ScriptableRenderContext.CreateShadowRendererList or ScriptableRenderContext.CreateSkyboxRendererList method.

### Static Properties

| Property | Description |
| --- | --- |
| nullRendererList | Returns an empty RendererList. |

### Properties

| Property | Description |
| --- | --- |
| isValid | Indicates whether the RendererList is valid or not. If the RendererList is valid, this returns true. Otherwise, this returns false. |
