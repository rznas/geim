### Description

Defines the way Unity chooses a probe to light a Renderer that is lit by Light Probes but positioned outside the bounds of the Light Probe tetrahedral hull.

Enable this option to require Unity to use the Ambient Probe to light a Renderer lit with Light Probes but positioned outside of a Light Probe tetrahedral hull.
