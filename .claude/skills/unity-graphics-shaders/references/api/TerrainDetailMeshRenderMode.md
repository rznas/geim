### Description

Options for determining the render mode of the mesh detail.

Changing the the mesh render mode alters how the detail is rendered.

 Additional resources: DetailRenderMode

### Properties

| Property | Description |
| --- | --- |
| VertexLit | Uses the vertex lit shader for rendering details |
| Grass | Uses the grass shader for rendering details |
