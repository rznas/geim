### Description

Provides an interface to the system that draws the Unity trial version and development build watermarks in screen space.

### Static Properties

| Property | Description |
| --- | --- |
| showDeveloperWatermark | Indicates and controls if the development build watermark should be drawn. |

### Static Methods

| Method | Description |
| --- | --- |
| IsVisible | Returns true if the watermark is visible on-screen. Whether the watermark is visible depends on your build settings and Unity plan. |
