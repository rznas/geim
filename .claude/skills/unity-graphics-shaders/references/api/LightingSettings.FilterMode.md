### Description

The available filtering modes for the Progressive Lightmapper.

### Properties

| Property | Description |
| --- | --- |
| None | No filtering. |
| Auto | The filtering is configured automatically. |
| Advanced | When enabled, you can configure the filtering settings for the Progressive Lightmapper. When disabled, the default filtering settings apply. |
