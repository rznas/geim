# VFX Project Settings

The VFX tab in the **Project Settings** window controls settings relevant to the Visual Effect Graph.

This tab is only visible if you install the Visual Effect Graph package. For information on how to install the Visual Effect Graph, see the packages documentation.

For information on the properties in this tab, see the Visual Effect Graph package documentation.
