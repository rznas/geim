### Description

Define a number of supported Rendering Layers on the Render Pipeline.

Render Pipeline supports 32 Rendering Layers by default.

### Properties

| Property | Description |
| --- | --- |
| maxRenderingLayersForPipeline | Define a maximum number of the supported Rendering Layers. |
| maxSupportedRenderingLayers | Extract a maximum number of supported Rendering Layers. |
