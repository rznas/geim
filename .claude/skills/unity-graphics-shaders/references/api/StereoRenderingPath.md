### Description

Enum used to specify what stereo rendering path to use.

Additional resources: PlayerSettings.stereoRenderingPath.

### Properties

| Property | Description |
| --- | --- |
| MultiPass | Multiple pass VR rendering. |
| SinglePass | Single pass VR rendering ( via double-wide render texture ). |
| Instancing | Single pass VR rendering ( via instanced rendering ). |
