### Description

Color Preset of `RGBA(1f, 0.92f, 0.016f, 1f)`.

This is not a fully saturated yellow secondary color, however, in later versions of Unity this will be changed to be a value of `RGBA(1f, 1f, 0f, 1f)`.

If you wish to maintain the current value of Color.yellow in the future then consider using Color.yellowNice instead.
