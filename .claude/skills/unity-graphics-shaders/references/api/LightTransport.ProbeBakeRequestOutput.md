### Description

A bitmask that describes the outputs to bake for a probe request.

### Properties

| Property | Description |
| --- | --- |
| RadianceDirect | Adds direct light contribution. |
| RadianceIndirect | Adds indirect light contribution. |
| Validity | Adds validity data. |
| Occlusion | Adds probe occlusion data. This data is used for Mixed lighting. |
| All | Adds all the data. |
