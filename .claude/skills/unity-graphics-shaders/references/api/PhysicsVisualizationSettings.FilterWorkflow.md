### Description

Decides whether the Workflow in the Physics Debug window should be `inclusive/` (ShowSelectedItems) or `exclusive` (HideSelectedItems).

By default HideSelectedItems is chosen and none of filtering items are selected. This means that nothing is hidden and thus everything is visible.

Additional resources: PhysicsVisualizationSettings.filterWorkflow.

### Properties

| Property | Description |
| --- | --- |
| HideSelectedItems | With HideSelectedItems you can hide selected filter items in order to easily discard uninteresting Collider properties thereby making overview and navigation easier. |
| ShowSelectedItems | With ShowSelectedItems and the Select None button is it easy to monitor very specific Colliders. |
