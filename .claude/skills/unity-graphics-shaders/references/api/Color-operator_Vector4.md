### Description

Colors can be implicitly converted to and from Vector4.

```csharp
using UnityEngine;public class Example : MonoBehaviour
{
    void Start()
    {
        Color newColor = new Vector4(0.3f, 0.4f, 0.6f);
    }
}
```
