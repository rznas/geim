### Description

The editor selected render mode for Scene View selection.

### Properties

| Property | Description |
| --- | --- |
| Hidden | The Renderer has no selection highlight or wireframe in the Editor. |
| Wireframe | The Renderer has wireframe but not selection highlight in the Editor. |
| Highlight | The Renderer has selection highlight but no wireframe in the Editor. |
