### Description

Unity raises this event when it loads a CustomRenderTexture.

Use this event to keep track of CustomRenderTexture loading. This is useful if you are writing your own CustomRenderTexture update loop.

Additional resources: Rendering.GraphicsSettings.disableBuiltinCustomRenderTextures.
