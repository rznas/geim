### Description

When this flag is set, Unity culls both eyes together for stereo rendering.

Default value: based on the Camera from which you obtained the CullingParameters.

Additional resources: ScriptableCullingParameters.cullingOptions, ScriptableRenderContext.Cull, CullingResults.
