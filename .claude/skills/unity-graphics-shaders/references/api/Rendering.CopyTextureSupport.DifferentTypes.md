### Description

Support for Graphics.CopyTexture between different texture types.

When this flag is set, Graphics.CopyTexture can do copies between different texture "dimensions", for example copy a single cubemap face into a regular 2D texture; or copy from a 2D texture into a slice of 2D Array texture. See Graphics.CopyTextureSupport for an overview.
