### Description

The particle GameObject filtering mode that specifies which objects are used by specific Particle System modules.

### Properties

| Property | Description |
| --- | --- |
| LayerMask | Include objects based on a layer mask, where all objects that match the mask are included. |
| List | Include objects based on an explicitly provided list. |
| LayerMaskAndList | Include objects based on both a layer mask and an explicitly provided list. |
