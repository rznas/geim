### Description

Alpha tested geometry uses this queue. This value corresponds to a render queue of 2450.

It’s a separate queue from Geometry one since it’s more efficient to render alpha-tested objects after all solid ones are drawn.

Additional resources: Material.renderQueue, Shader.renderQueue, subshader tags.
