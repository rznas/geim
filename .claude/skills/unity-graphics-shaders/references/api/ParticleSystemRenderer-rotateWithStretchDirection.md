### Description

Rotate the particles based on the direction they are stretched in. This is added on top of other particle rotation.

This property only has effect when freeformStretching is enabled. When freeformStretching is disabled, particles are always rotated based on the direction they are stretched in, even if rotateWithStretchDirection is false.
