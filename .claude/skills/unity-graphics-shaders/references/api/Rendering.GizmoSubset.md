### Description

Specifies whether gizmos render before or after postprocessing for a camera render.

### Properties

| Property | Description |
| --- | --- |
| PreImageEffects | Specifies gizmos that are rendered before ImageEffects. |
| PostImageEffects | Specifies gizmos that are rendered after ImageEffects. |
