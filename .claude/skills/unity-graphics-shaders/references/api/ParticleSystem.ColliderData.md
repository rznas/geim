### Description

Script interface for particle Collider data.

ColliderData contains information about which Colliders a particle is interacting with. It can be used in the MonoBehaviour.OnParticleTrigger callback, when the Trigger module is enabled.

### Public Methods

| Method | Description |
| --- | --- |
| GetCollider | Retrieve a specific Collider that a particle iss interacting with. |
| GetColliderCount | Returns how how many Colliders a particle is interacting with. |
