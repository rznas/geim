### Description

The unit of illuminance (lumen per square meter).

A light source that emits 1 lumen of luminous flux onto an area of 1 square meter has an illuminance of 1 lux.

Compatible light types: LightType.Directional, LightType.Box, LightType.Point, LightType.Spot, LightType.Pyramid.

Additional resources: Light.lightUnit, light component.
