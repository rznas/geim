### Description

Shadow resolution options for a Light.

Additional resources: Light.shadowResolution, QualitySettings.shadowResolution, Shadow map size calculation.

### Properties

| Property | Description |
| --- | --- |
| FromQualitySettings | Use resolution from QualitySettings (default). |
| Low | Low shadow map resolution. |
| Medium | Medium shadow map resolution. |
| High | High shadow map resolution. |
| VeryHigh | Very high shadow map resolution. |
