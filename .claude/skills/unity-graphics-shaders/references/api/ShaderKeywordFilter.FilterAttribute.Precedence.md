### Description

The precedence rules of shader keyword filter attributes that are active on the same keywords.

### Properties

| Property | Description |
| --- | --- |
| Normal | Unity ignores the shader keyword filter attribute if there's already an active attribute for the same keywords. |
| Override | Unity overrides any already active attributes on the same keywords with this shader keyword filter attribute. |
