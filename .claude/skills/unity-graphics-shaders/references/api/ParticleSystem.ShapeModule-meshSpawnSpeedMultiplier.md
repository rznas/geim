### Description

A multiplier of the Mesh spawn speed.

Changing this property is more efficient than accessing the entire curve, if you only want to change the overall speed multiplier. Additional resources: ParticleSystem.ShapeModule.meshSpawnMode, ParticleSystemShapeMultiModeValue, ParticleSystem.ShapeModule.shapeType.
