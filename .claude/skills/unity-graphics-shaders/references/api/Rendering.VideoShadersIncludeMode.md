### Description

Video shaders mode used by GraphicsSettings.

Additional resources: GraphicsSettings.videoShadersIncludeMode.

### Properties

| Property | Description |
| --- | --- |
| Never | Exclude video shaders from builds. This effectively disables video functionality. |
| Referenced | Include video shaders in builds when referenced by scenes. |
| Always | Include video shaders in builds (default). |
