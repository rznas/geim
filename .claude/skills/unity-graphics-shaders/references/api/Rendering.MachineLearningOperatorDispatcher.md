### Description

Static dispatcher class for executing machine learning operators.

### Static Methods

| Method | Description |
| --- | --- |
| Conv | Executes the convolution operator. |
| Gemm | Executes the general matrix multiplication (GEMM) operator. |
| Identity | Executes the identity operator. |
| Reduce | Executes the reduce operator. |
