### Description

Flag indicating if the collection is completely warmed up.

When a collection is "warmed up", all the GPU representations of the shader variants and graphics states in the collection will have been created. This is the case when completedWarmupCount is equal to totalGraphicsStateCount. Additional resources: WarmUp, completedWarmupCount.
