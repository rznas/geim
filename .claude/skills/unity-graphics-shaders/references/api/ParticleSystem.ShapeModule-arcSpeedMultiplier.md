### Description

A multiplier of the arc speed of the particle emission shape.

Changing this property is more efficient than accessing the entire curve, if you only want to change the overall speed multiplier. Additional resources: ParticleSystem.ShapeModule.arcMode, ParticleSystemShapeMultiModeValue, ParticleSystem.ShapeModule.shapeType.
