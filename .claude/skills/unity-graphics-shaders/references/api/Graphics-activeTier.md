### Description

The GraphicsTier for the current device.

**Note:** Graphics tiers are only supported in the Built-in Render Pipeline.

Unity auto-detects this value, and sets it during startup. This value determines which TierSettings are in use, and which tier variants Unity loads when it loads shaders. Changing this value affects any subsequently loaded shaders. For more information, see Graphics tiers.
