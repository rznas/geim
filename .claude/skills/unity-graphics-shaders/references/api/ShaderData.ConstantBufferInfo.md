### Description

Provides resource layout information about a shader constant buffer.

### Properties

| Property | Description |
| --- | --- |
| Fields | The fields in this constant buffer (Read Only). |
| Name | The name of this constant buffer (Read Only). |
| Size | The size (in bytes) of this constant buffer (Read Only). |
