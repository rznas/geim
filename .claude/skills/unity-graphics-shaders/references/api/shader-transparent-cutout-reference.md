# Transparent Cutout Shader Family reference

**Note.** Unity 5 introduced the Standard Shader which replaces these **shaders**.

The information in this section is a reference for the transparent cutout legacy shaders in Unity.

| **Topic** | **Description** |
| --- | --- |
| **Transparent Cutout Shader Family overview** | Reference for the transparent cutout legacy shaders in Unity, used for objects that have fully opaque and fully transparent parts. |
| **Transparent Cutout Vertex-Lit** | Reference for the legacy Transparent Cutout Vertex-Lit shader, which calculates lighting per-vertex. |
| **Transparent Cutout Diffuse** | Reference for the legacy Transparent Cutout Diffuse shader, which provides basic matte lighting. |
| **Transparent Cutout Specular** | Reference for the legacy Transparent Cutout Specular shader, which adds a specular highlight and uses the alpha channel for both transparent areas and the specular map. |
| **Transparent Cutout Bumped Diffuse** | Reference for the legacy Transparent Cutout Bumped Diffuse shader, which adds surface detail using a **normal map**. |
| **Transparent Cutout Bumped Specular** | Reference for the legacy Transparent Cutout Bumped Specular shader, which combines a normal map with a specular highlight and uses the alpha channel for both transparent areas and the specular map. |
