### Description

Options that represent the result of a ScriptableRenderContext.QueryRendererList operation.

### Properties

| Property | Description |
| --- | --- |
| kRendererListInvalid | The RendererList from the query operation is invalid. |
| kRendererListProcessing | Unity is still processing the RendererList. |
| kRendererListEmpty | There are no GameObjects in the current view that match the RendererList's criteria. |
| kRendererListPopulated | The RendererList is not empty. |
