### Description

Visible reflection probes sorting options.

Additional resources: CullingParameters.reflectionProbeSortingCriteria.

### Properties

| Property | Description |
| --- | --- |
| None | Do not sort reflection probes. |
| Importance | Sort probes by importance. |
| Size | Sort probes from largest to smallest. |
| ImportanceThenSize | Sort probes by importance, then by size. |
