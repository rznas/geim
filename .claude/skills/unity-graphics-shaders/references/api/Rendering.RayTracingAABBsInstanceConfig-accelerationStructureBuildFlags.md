### Description

The flags Unity uses when it builds the acceleration structure for the geometry referenced by this ray tracing instance configuration.

If you set `accelerationStructureBuildFlagsOverride` to `true`, Unity uses these flags and overrides the default build flags specified when creating a RayTracingAccelerationStructure.
