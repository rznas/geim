### Description

ReflectionProbeBlendInfo contains information required for blending probes.

Additional resources: Renderer.reflectionProbeUsage.

### Properties

| Property | Description |
| --- | --- |
| probe | Reflection Probe used in blending. |
| weight | Specifies the weight used in the interpolation between two probes, value varies from 0.0 to 1.0. |
