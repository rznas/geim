### Description

Scalar (R) render texture format, 32 bit floating point.

Note that not all graphics cards support floating point render textures. Use SystemInfo.SupportsRenderTextureFormat to check for support.

Additional resources: RenderTexture class, SystemInfo.SupportsRenderTextureFormat.
