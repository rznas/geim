### Description

Interface for when you extend the Lighting Explorer. Used in full overrides of the default behavior.

### Public Methods

| Method | Description |
| --- | --- |
| GetContentTabs | Returns the tabs that you have selected to display in the Lighting Explorer. |
| OnDisable | This is called when the Lighting Explorer OnDisable is called, or when you switch to another extension. |
| OnEnable | This is called when the Lighting Explorer OnEnable is called, or when you switch to another extension. |
