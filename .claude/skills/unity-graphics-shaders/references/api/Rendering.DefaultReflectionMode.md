### Description

Default reflection mode.

Unity can use a custom texture or generate a specular reflection texture from the skybox.

Additional resources: RenderSettings.defaultReflectionMode, Lighting Window.

### Properties

| Property | Description |
| --- | --- |
| Skybox | Skybox-based default reflection. |
| Custom | Custom default reflection. |
