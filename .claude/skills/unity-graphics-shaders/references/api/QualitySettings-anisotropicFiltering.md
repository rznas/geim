### Description

Global anisotropic filtering mode.

```csharp
using UnityEngine;public class Example : MonoBehaviour
{
    void Start()
    {
        // Disable all anisotropic filtering
        QualitySettings.anisotropicFiltering = AnisotropicFiltering.Disable;
    }
}
```
