### Description

The mode used for selecting rows of an animation in the Texture Sheet Animation Module.

### Properties

| Property | Description |
| --- | --- |
| Custom | Use a specific row for all particles. |
| Random | Use a random row for each particle. |
| MeshIndex | Use the mesh index as the row, so that meshes can be mapped to specific animation frames. |
