### Description

Desired color space (Read Only).

This is the desired color space based on player settings (PlayerSettings.colorSpace). Note that if the platform or hardware does not support the given color space, the actually used color space might end up being different. Use activeColorSpace to check for that.

Additional resources: Linear and Gamma rendering, ColorSpace, activeColorSpace.
