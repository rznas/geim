### Description

When enabled, you can configure the filtering settings for the Progressive Lightmapper. When disabled, the default filtering settings apply.

In Advanced mode, you can manually configure the selected filter kernel for individual lightmap targets: Ambient Occlusion, direct lighting, and indirect lighting.
