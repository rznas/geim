# Creating a hierarchy of materials

To create a hierarchy of materials that uses property inheritance, use material variants. If you create hundreds of materials, material variants are easier to manage than duplicating materials.

| **Topic** | **Description** |
| --- | --- |
| Material variants | Learn about the benefits and limitations of using material variants to manage materials. |
| Create a material variant | Create a material variant, change its parent, or convert a material variant back to a material. |
| Control material variant properties | Propagate property changes from a parent to its children, or override parent properties. |
