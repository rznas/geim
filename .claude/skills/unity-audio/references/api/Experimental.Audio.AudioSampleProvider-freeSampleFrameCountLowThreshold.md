### Description

When the free sample count falls below this threshold, the AudioSampleProvider.sampleFramesAvailable event and associated native is emitted.

The associated native event handler, set with Experimental.Audio.AudioSampleProvider.SetSampleFramesAvailableNativeEventHandler is also invoked.
