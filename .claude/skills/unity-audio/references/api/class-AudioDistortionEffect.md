# Audio Distortion Effect

The **Distortion Effect** distorts the sound from an AudioMixer group.

## Properties

| ***Property:*** | ***Function:*** |
| --- | --- |
| **Distortion** | Distortion value. 0.0 to 1.0. Default = 0.5. |

## Details

Apply the **Distortion Effect** to simulate the sound of a low quality radio transmission.
