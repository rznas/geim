# Audio Distortion Filter

The **Audio Distortion Filter** distorts the sound from an Audio Source or sounds that reach the Audio Listener.

## Properties

| ***Property:*** | ***Function:*** |
| --- | --- |
| **Distortion Level** | Distortion value to apply to the sound. Value ranges from 0.0 to 1.0. Default is 0.5. |

## Details

Apply the **Audio Distortion Filter** to simulate the sound of a low quality radio transmission.
