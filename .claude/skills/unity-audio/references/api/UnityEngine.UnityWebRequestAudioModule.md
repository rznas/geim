### Description

The UnityWebRequestAudio module provides the DownloadHandlerAudioClip class to use UnityWebRequest to download AudioClips.

### Classes

| Class | Description |
| --- | --- |
| DownloadHandlerAudioClip | A DownloadHandler subclass specialized for downloading audio data for use as AudioClip objects. |
| UnityWebRequestMultimedia | Helpers for downloading multimedia files using UnityWebRequest. |
