# Audio filters

Use **Audio Effects** to filter the frequency ranges of audio or apply reverb effects. You can use Audio Effects on Audio Source components and Audio Listener components.

| **Topic** | **Description** |
| --- | --- |
| **Use audio filters** | Learn how to apply and order **audio filters** on Audio Sources and Listeners. |
| **Audio Low Pass Filter** | Reference for the Audio Low Pass filter, which passes low frequencies and removes high frequencies. |
| **Audio High Pass Filter** | Reference for the Audio High Pass filter, which passes high frequencies and removes low frequencies. |
| **Audio Echo Filter** | Reference for the Audio Echo filter, which repeats a sound after a given delay. |
| **Audio Distortion Filter** | Reference for the Audio Distortion filter, which applies a **distortion effect** to audio signals. |
| **Audio Reverb Filter** | Reference for the Audio Reverb filter, which applies a customized reverb effect to audio signals. |
| **Audio Chorus Filter** | Reference for the Audio Chorus filter, which processes audio signals to create a chorus effect. |
