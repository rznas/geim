### Description

A struct that contains packing information of the sprite to Atlas.

### Properties

| Property | Description |
| --- | --- |
| page | The page to which the sprite is packed into SpriteAtlas. |
| rot | The rotation of the Sprite when packed into SpriteAtlas. |
| x | Location in x axis to where the Sprite is packed to Texture. |
| y | Location in y axis to where the Sprite is packed to Texture. |
