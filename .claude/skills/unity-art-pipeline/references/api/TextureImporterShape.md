### Description

The shape of the imported texture.

The default value is Texture2D. The texture can also be imported as a Cubemap, Texture2DArray or a Texture3D.

Additional resources: TextureImporterSettings.textureShape, Texture Assets.

### Properties

| Property | Description |
| --- | --- |
| Texture2D | Import the texture as a 2D texture (default). |
| TextureCube | Import the texture as a Cubemap. |
| Texture2DArray | Import the texture as a 2D Array texture. |
| Texture3D | Import the texture as a 3D texture. |
