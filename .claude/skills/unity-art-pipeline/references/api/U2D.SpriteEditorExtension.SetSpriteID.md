### Parameters

| Parameter | Description |
| --- | --- |
| sprite | The Sprite to set. |
| guid | The GUID to set for the Sprite. |

### Description

Sets a Sprite's Global Unique Identifier (GUID) for easy identification later.

During Sprite Asset generation, you can manually set a Sprite's SpriteID to identify the Sprite from the data that was used to generate it.

This is usually needed when Sprites are generated by ScriptedImporter via Sprite.Create and the ScriptedImporter wants to support SpriteEditorModuleBase that uses AssetPostprocessor to modify during Sprite generation.
