### Description

The mode of the Animator's recorder.

The recorder can either be Offline, in Playback or in Record.

### Properties

| Property | Description |
| --- | --- |
| Offline | The Animator recorder is offline. |
| Playback | The Animator recorder is in Playback. |
| Record | The Animator recorder is in Record. |
