### Description

The maximum value of each sample in the detail map of the terrain data.

This value will depend on the set DetailScatterMode. In DetailScatterMode.CoverageMode, values of up to 255 are stored. In DetailScatterMode.InstanceCountMode, values of up to 16 are stored.
