### Description

This property, together with indexCount, determines which part of the whole index buffer is used by a particular Mesh LOD. 

The index start is relative to the sub-mesh holding the LOD. This should be a positive number or zero.

Additional resources: indexCount, Mesh.SetLod, Mesh.SetLods.
