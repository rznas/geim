### Description

Smoothing angle (in degrees) for calculating normals.

How sharp the edge has to be to be turned into a hard edge when calculating normals. This setting is only used when importNormals is set to ModelImporterNormals.Calculate.

Additional resources: importNormals, ModelImporterNormals.
