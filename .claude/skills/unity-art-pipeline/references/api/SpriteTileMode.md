### Description

Tiling mode for SpriteRenderer.tileMode.

### Properties

| Property | Description |
| --- | --- |
| Continuous | Sprite Renderer tiles the sprite continuously when is set to SpriteRenderer.tileMode. |
| Adaptive | Sprite Renderer tiles the sprite once the Sprite Renderer size is above SpriteRenderer.adaptiveModeThreshold. |
