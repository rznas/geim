### Description

This is a Texture2D generated by TextureGenerator.GenerateTexture based on the colorBuffer argument passed to that function. If the generator is configured to generate a non-2D texture (cubemap, 3D, ...), this member will always be null. To get the result for an arbitrary texture type, please use the TextureGenerationOutput.output property. We recommend always using TextureGenerationOutput.output as the texture property will deprecated/removed in the future.
