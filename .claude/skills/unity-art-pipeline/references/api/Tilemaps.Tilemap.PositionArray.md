### Description

A read-only array containing Tilemap positions.

### Properties

| Property | Description |
| --- | --- |
| Length | The number of elements in the PositionArray. |
| this[int] | Returns the Vector3Int position indexed in the PositionArray. |

| Method | Description |
| --- | --- |
