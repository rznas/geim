### Description

Mesh compression options for ModelImporter.

Compressing meshes saves space in the built game, but more compression introduces more artifacts in vertex data.

Additional resources: ModelImporter.meshCompression.

### Properties

| Property | Description |
| --- | --- |
| Off | No mesh compression (default). |
| Low | Low amount of mesh compression. |
| Medium | Medium amount of mesh compression. |
| High | High amount of mesh compression. |
