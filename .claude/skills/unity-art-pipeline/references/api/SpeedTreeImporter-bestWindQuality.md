### Description

Returns the best-possible wind quality on this asset (configured in SpeedTree modeler).

The possible wind quality values are as follows:

`0` - Off
 `1` - Fastest
 `2` - Fast
 `3` - Better
 `4` - Best
 `5` - Palm (only available on palm-like trees)
