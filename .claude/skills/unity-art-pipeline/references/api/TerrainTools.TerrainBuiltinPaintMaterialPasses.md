### Description

Built-in render passes for paint material.

### Properties

| Property | Description |
| --- | --- |
| RaiseLowerHeight | Built-in render pass for raising and lowering Terrain height. |
| StampHeight | Built-in render pass for stamping heights on the Terrain. |
| SetHeights | Built-in render pass for setting Terrain height. |
| SmoothHeights | Built-in render pass for smoothing the Terrain height. |
| PaintTexture | Built-in render pass for painting the splatmap texture. |
| PaintHoles | Built-in render pass for painting Terrain holes. |
