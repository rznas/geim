### Description

Normal generation options for ModelImporter.

Normals can either be imported from the Model file (default), calculated by Unity, or not included in the imported Mesh at all. Vertex normals are most often used for real-time lighting.

Additional resources: ModelImporter.importNormals, ModelImporter.importBlendshapeNormals, ModelImporterTangents, Mesh.normals.

### Properties

| Property | Description |
| --- | --- |
| Import | Import vertex normals from model file (default). |
| Calculate | Calculate vertex normals. |
| None | Do not import vertex normals. |
