### Description

Sets the playback position in the recording buffer.

When in playback mode (see StartPlayback), this value is used for controlling the current playback position in the buffer (in seconds). The value can range between recorderStartTime and recorderStopTime Additional resources: StartPlayback, StopPlayback.
