### Description

Contains a set of typed properties for describing a texture input of a MaterialDescription.

### Properties

| Property | Description |
| --- | --- |
| offset | UV coordinates offset (x,y). |
| path | Absolute path to the texture asset. |
| relativePath | Path to the texture asset relative to the Asset folder. |
| scale | UV coordinates scaling (x,y). |
| texture | Reference to the texture asset. |
