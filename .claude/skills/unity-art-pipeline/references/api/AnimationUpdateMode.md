### Description

The update mode of the Animation component.

### Properties

| Property | Description |
| --- | --- |
| Normal | Animation updates in the Update loop. |
| Fixed | Animation updates in the FixedUpdate loop. Use this mode to evaluate animation independent of frame rate. |
