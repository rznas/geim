### Parameters

| Parameter | Description |
| --- | --- |
| name | The name of the sprite atlas also used as Tag. |
| downScaleMultiplier | Runtime configuration for the sprite atlas, including scale multiplier. |
| pages | Array of atlas pages containing sprites and packed textures. |

### Description

Creates a new sprite atlas with the given name and runtime configuration and binds sprites to textures.
