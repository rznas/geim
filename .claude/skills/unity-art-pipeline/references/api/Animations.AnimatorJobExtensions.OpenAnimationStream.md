### Parameters

| Parameter | Description |
| --- | --- |
| animator | The Animator instance that calls this method. |
| stream | The new stream. |

### Returns

**bool** Returns whether or not the stream has been opened.

### Description

Open a new stream on the Animator.

The stream opened this way must be closed using CloseAnimationStream.
