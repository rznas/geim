### Description

Defines Cubemap convolution mode.

This is useful if you need to pre-compute a diffuse or glossy specular reflections in the texture.

Additional resources: TextureImporterSettings.cubemapConvolution.

### Properties

| Property | Description |
| --- | --- |
| None | No convolution needed. This Cubemap texture represents mirror reflection or Skybox. |
| Specular | Specular convolution (aka Prefiltered Environment Map). |
| Diffuse | Diffuse convolution (aka irradiance Cubemap). |
