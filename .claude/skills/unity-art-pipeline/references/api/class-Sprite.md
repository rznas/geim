# Sprite asset reference

Explore the properties of a **sprite** asset. Unity displays the properties in the ****Inspector**** window when you select a sprite asset in the **Project** window.

For more information, refer to Create sprites from a texture.

| **Property** | **Description** |
| --- | --- |
| **Name** | Sets the name of the sprite. |
| **Pivot** | Sets the position of the point Unity uses for transformations such as rotation. The values are normalized, so they range from 0, 0 at the bottom-left of the sprite to 1, 1 at the top-right of the sprite. |
| **Border** | Sets the size of the border Unity uses for 9-slicing. The properties are:   **L**: The width of the left border in pixels.  **R**: The width of the right border in pixels.  **T**: The height of the top border in pixels.  **B**: The height of the bottom border in pixels. |
