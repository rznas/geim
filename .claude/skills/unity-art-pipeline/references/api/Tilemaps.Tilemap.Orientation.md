### Description

Determines the orientation of Tiles in the Tilemap.

### Properties

| Property | Description |
| --- | --- |
| XY | Orients Tiles in the XY plane. |
| XZ | Orients Tiles in the XZ plane. |
| YX | Orients Tiles in the YX plane. |
| YZ | Orients Tiles in the YZ plane. |
| ZX | Orients Tiles in the ZX plane. |
| ZY | Orients Tiles in the ZY plane. |
| Custom | Use a custom orientation to all tiles in the tile map. |
