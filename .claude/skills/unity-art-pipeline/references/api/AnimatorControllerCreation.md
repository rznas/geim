# Create an Animator Controller

You can view and set up character behavior from the **Animator Controller** view (Menu: **Window** > **Animation** > **Animator**).

To create an Animator Controller, do one of the following:

- Select **Create** > **Animator Controller** from the Project window.
- Right-click in the Project window and select **Create** > **Animator Controller**.
- From the Assets menu, select **Assets** > **Create** > **Animator Controller**.

This creates a `.controller` asset. In the **Project Browser** window.

After the **state machine** setup has been made, you can drop the controller onto the **Animator component** of any character with an **Avatar** in the **Hierarchy View**.
