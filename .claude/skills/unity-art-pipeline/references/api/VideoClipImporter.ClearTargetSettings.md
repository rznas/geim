### Parameters

| Parameter | Description |
| --- | --- |
| platform | Platform name. |

### Description

Clear the platform-specific import settings for the specified platform, causing them to go back to the default settings.

See VideoClipImporter.defaultTargetSettings. The options for the platform string are "Standalone", "Android", "iOS", "Lumin", "PS4", "Switch", "tvOS", "WebGL", "WSA", or "XboxOne".
