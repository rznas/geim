### Description

Returns whether the TilemapRenderer automatically detects the bounds to extend chunk culling by.

### Properties

| Property | Description |
| --- | --- |
| Auto | The TilemapRenderer will automatically detect the bounds of extension by inspecting the Sprite/s used in the Tilemap. |
| Manual | The user adds in the values used for extend the bounds for culling of Tilemap chunks. |
