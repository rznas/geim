### Description

Data that describes the important points of the shape.

### Properties

| Property | Description |
| --- | --- |
| leftTangent | The position of the left tangent in local space. |
| mode | The various modes of the tangent handles. They could be continuous or broken. |
| position | The position of this point in the object's local space. |
| rightTangent | The position of the right tangent point in the local space. |
