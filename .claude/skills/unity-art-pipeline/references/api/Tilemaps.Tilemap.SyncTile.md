### Description

A Struct for containing changes to a Tile when it has been changed on a Tilemap.

### Properties

| Property | Description |
| --- | --- |
| position | The position of the Tile on a Tilemap which has changed. |
| tile | The Tile at the given position on the Tilemap. |
| tileData | The properties of the Tile at the given position on the Tilemap. |
