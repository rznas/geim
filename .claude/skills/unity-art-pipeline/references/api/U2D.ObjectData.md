### Description

ObjectData holds information about a packed Object such as Sprite in the sprite atlas.

### Properties

| Property | Description |
| --- | --- |
| asset | Entity ID of the sprite. |
| packInfo | Packing information for this object. X, Y coordinates of the packed position. |
