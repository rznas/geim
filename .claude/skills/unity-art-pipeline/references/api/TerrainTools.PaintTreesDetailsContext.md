### Description

Represents a context object for information used when scattering trees and detail objects across terrains.

### Properties

| Property | Description |
| --- | --- |
| neighborTerrains | Neighboring terrain objects. |
| neighborUvs | Neighboring terrain UVs. |

### Static Methods

| Method | Description |
| --- | --- |
| Create | Initializes and returns a scattering context object. |
