### Description

Represents the position and Tile information to change in a Tilemap.

### Properties

| Property | Description |
| --- | --- |
| color | The color to set the Tile to. |
| position | The position to change Tile properties at. |
| tile | The Tile to set on the Tilemap. |
| transform | The transform matrix to set the Tile to. |
