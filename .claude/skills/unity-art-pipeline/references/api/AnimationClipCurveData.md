### Description

An AnimationClipCurveData object contains all the information needed to identify a specific curve in an AnimationClip. The curve animates a specific property of a component / material attached to a game object / animated bone.

Additional resources: AnimationUtility, ObjectNames.

### Properties

| Property | Description |
| --- | --- |
| curve | The actual animation curve. |
| path | The path of the game object / bone being animated. |
| propertyName | The name of the property being animated. |
| type | The type of the component / material being animated. |
