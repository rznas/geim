### Description

SpriteShapeSegment contains info of sub-meshes generated by the SpriteShape generator C# Job later fed to SpriteShapeRenderer.

### Properties

| Property | Description |
| --- | --- |
| geomIndex | Geometry index of list of sub-meshes generated by SpriteShape. |
| indexCount | Index count of SpriteShape sub-mesh. |
| spriteIndex | Index of sprite that is used to generate segment/corner for this SpriteShapeSegment. |
| vertexCount | Vertex count of SpriteShape sub-mesh. |
