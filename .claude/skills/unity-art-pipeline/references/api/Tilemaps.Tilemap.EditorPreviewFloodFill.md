### Parameters

| Parameter | Description |
| --- | --- |
| position | Start position of the flood fill on the Tilemap. |
| tile | TileBase to place. |

### Description

Does an editor preview of a flood fill with the given Tile to place. on the Tilemap starting from the given coordinates.

The editor preview is used for previewing possible additions to the Tilemap. Refer to Scriptable Tiles and Tilemap for more information.
