### Description

Wrap mode for Playables.

### Properties

| Property | Description |
| --- | --- |
| Hold | Hold the last frame when the playable time reaches it's duration. |
| Loop | Loop back to zero time and continue playing. |
| None | Do not keep playing when the time reaches the duration. |
