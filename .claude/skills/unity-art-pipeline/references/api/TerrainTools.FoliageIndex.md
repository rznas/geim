### Description

The index at which you should place the foliage tools in the Terrain Tools overlay.

### Properties

| Property | Description |
| --- | --- |
| PaintDetails | The index at which you should place the Paint Details foliage tool in the Terrain Tools overlay. |
| PaintTrees | The index at which you should place the Paint Trees foliage tool in the Terrain Tools overlay. |
