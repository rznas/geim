### Description

Sets an Atlas to be the Master Atlas.

This method sets a valid Master Atlas to this Atlas but does not automatically turn it into a Variant Sprite Atlas. Use SpriteAtlasExtensions.SetIsVariant to turn this Atlas into a Variant.
