### Description

IK Hint.

Used to set and get IK weights and position. Additional resources: Animator.GetIKHintPosition, Animator.GetIKHintPositionWeight, Animator.SetIKHintPosition, and Animator.SetIKHintPositionWeight.

### Properties

| Property | Description |
| --- | --- |
| LeftKnee | The left knee IK hint. |
| RightKnee | The right knee IK hint. |
| LeftElbow | The left elbow IK hint. |
| RightElbow | The right elbow IK hint. |
