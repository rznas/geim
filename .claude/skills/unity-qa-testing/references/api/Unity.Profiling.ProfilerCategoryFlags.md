### Description

Options for determining if a Profiler category is built into Unity by default.

### Properties

| Property | Description |
| --- | --- |
| None | Use this flag to determine that a Profiler category is not built into Unity by default. |
| Builtin | Use this flag to determine that a Profiler category is built into the Unity Editor by default. |
