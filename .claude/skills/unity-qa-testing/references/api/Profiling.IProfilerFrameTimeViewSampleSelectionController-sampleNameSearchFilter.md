### Description

This filters the samples displayed in Hierarchy view to only include the names that include this string.

Note: Only the Hierarchy and Raw Hierarchy views currently implement a search filter. Timeline view is not affected by setting this value. When this value is set while in Timeline view, the search is applied to a Hierarchy view as soon as it is shown again.
