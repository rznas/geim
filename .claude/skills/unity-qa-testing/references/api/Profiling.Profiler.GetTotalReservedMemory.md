### Description

Returns the amount of reserved system memory.

Unity requests the amount of memory that the game will use. GetTotalReservedMemory is the full amount that Unity currently has. Part of this memory is used by the game using GetTotalAllocatedMemory, and the remainer is stored and unused.

Additional resources: GetTotalAllocatedMemory, GetTotalUnusedReservedMemory.
