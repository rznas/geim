### Description

Use to indicate that ProfilerRecorder should collect GPU timing instead of CPU.

If you use ProfilerRecorderOptions.GpuRecorder to create the recorder, make sure you set the MarkerFlags.SampleGpu flag for the ProfilerMarker it is used with.
