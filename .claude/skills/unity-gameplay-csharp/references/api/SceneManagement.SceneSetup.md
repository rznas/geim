### Description

The setup information for a Scene in the SceneManager. This cannot be used in Play Mode.

### Properties

| Property | Description |
| --- | --- |
| isActive | If the Scene is active. |
| isLoaded | If the Scene is loaded. |
| path | Path of the Scene. Should be relative to the project folder. Like: "Assets/MyScenes/MyScene.unity". |
