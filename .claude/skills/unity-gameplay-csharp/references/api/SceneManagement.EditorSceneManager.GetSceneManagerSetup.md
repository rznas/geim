### Returns

**SceneSetup[]** An array of SceneSetup classes - one item for each Scene.

### Description

Returns the current setup of the SceneManager.

The returned array contains SceneSetup instances, one for each Scene. These indicate the active state, loaded state, and path of each of the current Scenes in the SceneManager.
