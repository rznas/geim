### Returns

**Stage** The current Stage that is currently being rendered in the Scene view and shown in the Hierarchy.

### Description

The current Stage can either be the MainStage or any other opened Stage, visualized in the Scene view as the last breadcrumb in the breadcrumb bar.

Additional resources: StageUtility.GetMainStage, StageUtility.GetStage.
