### Description

When a Component or GameObject is matched with objects in the Prefab Asset then existing values can be recorded as overrides on the new Prefab instance if this property is set to true.

If the values from the Prefab Asset is wanted instead then set this property to false. Additional resources: componentsNotMatchedBecomesOverride, gameObjectsNotMatchedBecomesOverride.
