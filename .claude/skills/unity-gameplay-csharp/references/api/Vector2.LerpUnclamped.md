### Description

Linearly interpolates between vectors `a` and `b` by `t`.

When `t` = 0 returns `a`. 
 When `t` = 1 return `b`. 
 When `t` = 0.5 returns the midpoint of `a` and `b`.

Additional resources: Lerp.
