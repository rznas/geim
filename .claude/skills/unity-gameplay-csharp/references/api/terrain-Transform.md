# Transform terrain features

**Note:** These tools are only available in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

The transform tools move existing **terrain** features relative to the brush: away or towards it, along its path, or around its center.

| Tool | Description |
| --- | --- |
| Pinch or spread out the terrain | Pinch elevated parts of the terrain to a central point, or spread them away from that point. |
| Smudge the terrain | Move elevated parts of the terrain along the path of the brush stroke. |
| Twist the terrain | Rotate elevated parts of the terrain around a central point. |
