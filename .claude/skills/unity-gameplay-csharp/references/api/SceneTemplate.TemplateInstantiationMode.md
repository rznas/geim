### Description

An enumeration of options for handling a Scene dependency Asset when you instantiate a SceneTemplateAsset.

### Properties

| Property | Description |
| --- | --- |
| Clone | Specifies that the dependency Asset is cloned when the SceneTemplateAsset is instantiated. |
| Reference | Specifies that the dependency Asset is kept as a reference when the SceneTemplateAsset is instantiated. |
