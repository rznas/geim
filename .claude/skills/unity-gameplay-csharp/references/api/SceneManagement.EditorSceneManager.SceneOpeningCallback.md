### Parameters

| Parameter | Description |
| --- | --- |
| path | Path of the Scene to be opened. This is relative to the Project path. |
| mode | Mode that is used when opening the Scene. |

### Description

Callbacks of this type which have been added to the sceneOpening event are called just before opening a Scene.
