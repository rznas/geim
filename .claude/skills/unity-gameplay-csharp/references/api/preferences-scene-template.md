# Scene Template preferences reference

Customize these settings to control the behavior of **scene** templates in the Unity Editor.

To open these preferences, go to **Edit** > **Preferences** > **Scene Template** (macOS: **Unity** > **Settings** > **Scene Template**).

The **Scene Template** section contains the following settings.

| **Property** | **Description** |
| --- | --- |
| **Default Scene** | Sets which scene opens when no other scenes were previously opened. The following options are available:  **Scene Template**: Opens the default scene template.  **Built-in Scene**: Opens Unity’s built-in default scene. This is the default setting. |
