### Description

Event which is called when the bake job calls InputExtraction.ExtractFromScene. This happens when a bake is started from the Editor or a bake is started with the lightmapping API. Additional resources: Lightmapping.Bake, Lightmapping.BakeAsync.

Additional resources: InputExtractionFunction.
