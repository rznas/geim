### Description

An enumerator that allows you to iterate over all direct children of a transform handle.

### Properties

| Property | Description |
| --- | --- |
| Current | Current enumerated transform handle. |

### Public Methods

| Method | Description |
| --- | --- |
| Dispose | Disposes the enumerator. |
| MoveNext | Moves the enumerator to the next transform handle. |
