# Smudge the terrain

**Note:** This tool is only available in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

To move elevated parts of the **terrain** along the path of the brush stroke, use the Smudge tool.

**Tip:** The Smudge and Pinch tools can appear similar: When the Pinch tool’s **Pinch Amount** is negative, it pushes the terrain away. This is indeed similar to the Smudge tool, but the Smudge tool doesn’t always create a smooth surface.

## Access the tool

To smudge the terrain, in the Terrain tile’s **Inspector** window, select the **Paint Terrain** tool > tool dropdown > **Transform** > **Smudge**.

## Tool options

The Smudge tool is brush-based. To learn about working with brushes, refer to Brushes.

## Smudge controls

Use the **Targets** selection to decide whether the Smudge tool impacts only the material, only the height, or both.

**Tip**: The change in the materials is only visible if there are at least two materials in the smudge area.
