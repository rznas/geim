### Description

Isolates a GameObject and its descendants.

If a GameObject is already in Isolation mode when calling this method, the isolated GameObject is hidden.

### Description

Isolates an Array of GameObjects and their descendants.

If a GameObject is already in Isolation mode when calling this method, the isolated GameObject is hidden.
