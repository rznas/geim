### Parameters

| Parameter | Description |
| --- | --- |
| value | Previous active Scene and the new active Scene. |

### Description

Subscribe to this event to get notified when the active Scene has changed.

This event is sent in both Play mode and Edit mode. If the event is only needed during Play mode then use SceneManager.activeSceneChanged.
