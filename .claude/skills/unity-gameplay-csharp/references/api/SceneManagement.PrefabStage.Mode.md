### Description

Specifies the mode that Unity uses to open a Prefab stage. You can open the stage either in isolation or in context.

You can only open a Prefab stage in context if you open it via an instance of the Prefab Asset.

### Properties

| Property | Description |
| --- | --- |
| InIsolation | Editing in isolation means that you can edit a Prefab Asset on its own in Prefab Mode. |
| InContext | Editing in context means that you can edit a Prefab Asset in Prefab Mode and still retain the context of the Prefab instance you opened through. |
