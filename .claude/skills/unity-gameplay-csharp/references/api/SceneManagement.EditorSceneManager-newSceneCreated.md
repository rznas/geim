### Description

This event is called after a new Scene has been created.

This can be useful to add your own default GameObjects to a Scene.

Use the setup argument to determine if any standard objects have already been added to the Scene or if the Scene is empty.

Use the mode to determine if the Scene was added additively or if it is a new single Scene.

Additional resources: NewSceneCreatedCallback.
