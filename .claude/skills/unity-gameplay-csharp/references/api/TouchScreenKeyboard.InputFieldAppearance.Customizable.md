### Description

Customize the visibility of the input field based on the requirement of your application.

Use TouchScreenKeyboard.hideInput to control the visibility of the input field. TouchScreenKeyboard.hideInput has an effect only if TouchScreenKeyboard.inputFieldAppearance returns `Customizable`.
