### Description

Gets whether the `distance` represents an overlap or not.

The distance can be positive indicating that the Collider2D are separated (not overlapped), zero indicating that they are touching (but not overlapped) or negative indicating that they are overlapped.

This property is therefore a check to see if distance is negative which indicates the Collider2D are overlapped.
