### Description

Describes a contact where two shapes collide.

These contact points are generated by the ImmediatePhysics.GenerateContacts function.

### Properties

| Property | Description |
| --- | --- |
| Normal | The normal of the contact point. |
| Point | The position of the contact point in world space. |
| Separation | The distance between the shapes at the contact point. |
