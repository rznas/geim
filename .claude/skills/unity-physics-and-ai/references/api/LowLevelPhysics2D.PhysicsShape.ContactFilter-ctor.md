### Parameters

| Parameter | Description |
| --- | --- |
| categories | A PhysicsMask defining the categories this object is in. |
| contacts | A PhysicsMask defining the categories this object will produce contacts with. |
| groupIndex | The override group this filter belongs to with zero indicating no group. Groups allow a certain group of objects to never collide (negative) or always collide (positive). |

Create a contact filter.
