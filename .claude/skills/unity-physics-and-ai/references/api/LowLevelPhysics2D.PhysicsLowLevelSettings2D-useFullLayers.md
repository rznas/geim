### Description

Controls if full 64-bit layers are used based upon PhysicsLowLevelSettings2D.physicsLayerNames or if not, the standard 32-bit layers based upon UnityEngine.LayerMask. If a PhysicsLowLevelSettings2D asset is assigned then the full layers (PhysicsLowLevelSettings2D.physicsLayerNames) will be used if PhysicsLowLevelSettings2D.useFullLayers is also active. If no PhysicsLowLevelSettings2D asset is assigned then the global layers (See UnityEngine.LayerMask) will be used.
