### Description

Enables the links generation for this object.

When build sources are obtained using NavMeshBuilder.CollectSources, this value can be affected by using NavMeshBuildMarkup. If this value is `true` and links parameters are valid in NavMeshBuildSettings this source will be considered during links generation.
