### Parameters

| Parameter | Description |
| --- | --- |
| position | The 3D position to extract the element depth from. |

### Description

Set the element depth using the specified 3D position. The relevant axis will be extracted using the current PhysicsWorld.transformPlane. This is only used in the Unity Editor or in a Development Player.

 For more details, see PhysicsWorld.elementDepth.
