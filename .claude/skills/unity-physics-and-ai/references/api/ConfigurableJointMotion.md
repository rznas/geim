### Description

Constrains movement for a ConfigurableJoint along the 6 axes.

### Properties

| Property | Description |
| --- | --- |
| Locked | Motion along the axis will be locked. |
| Limited | Motion along the axis will be limited by the respective limit. |
| Free | Motion along the axis will be completely free and completely unconstrained. |
