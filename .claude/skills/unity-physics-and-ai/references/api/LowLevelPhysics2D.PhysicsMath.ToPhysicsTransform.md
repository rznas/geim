### Parameters

| Parameter | Description |
| --- | --- |
| transform | The 3D transform to use. |
| transformPlane | The transform plane to use. |

### Returns

**PhysicsTransform** The 2D transform.

### Description

Transform a 3D UnityEngine.Transform position and rotation to a 2D PhysicsTransform. Scale is not part of a PhysicsTransform therefore it is ignored.
