### Description

Controls what aspects of is drawn using LowLevelPhysics2D.PhysicsWorld.Draw.

### Properties

| Property | Description |
| --- | --- |
| Interior | The interior of the area is drawn. |
| Outline | The outline of the area is drawn. |
| Orientation | The orientation of the area is drawn (if applicable). This is only drawn if the Outline is drawn. |
| All | A combination drawn of: PhysicsWorld.DrawFillOptions.InteriorPhysicsWorld.DrawFillOptions.OutlinePhysicsWorld.DrawFillOptions.Orientation |
