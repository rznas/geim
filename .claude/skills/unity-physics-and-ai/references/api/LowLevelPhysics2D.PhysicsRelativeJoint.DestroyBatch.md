### Parameters

| Parameter | Description |
| --- | --- |
| joints | The joints to destroy. |

### Description

Destroy a batch of joints. Owned joints will produce a warning and will not be destroyed (see PhysicsJoint.SetOwner). Any invalid joints will be ignored.
