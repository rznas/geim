### Description

The set of basic geometry shape types that can exist.

Additional resources: BoxGeometry, SphereGeometry, CapsuleGeometry, ConvexMeshGeometry, TriangleMeshGeometry, TerrainGeometry.

### Properties

| Property | Description |
| --- | --- |
| Sphere | A sphere shape. |
| Capsule | A capsule shape. |
| Box | A cube shape. |
| ConvexMesh | A convex mesh shape. |
| TriangleMesh | A triangle or non-convex mesh shape. |
| Terrain | The geometric shape of a Terrain collider. |
| Invalid | An invalid shape type. |
