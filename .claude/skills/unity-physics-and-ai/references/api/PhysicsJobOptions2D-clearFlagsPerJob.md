### Description

Controls the minimum number of flags to be cleared in each simulation job.

For an overview of these properties and advice on how to use them, see PhysicsJobOptions2D.

The following job profiler names are affected by this property:

• Physics2D.ClearContactIslandFlagsJob
 • Physics2D.ClearBodyIslandFlagsJob.
