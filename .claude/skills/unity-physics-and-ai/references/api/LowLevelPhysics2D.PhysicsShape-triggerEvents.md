### Description

Controls whether this shape produces triggers events which can be retrieved after the simulation has completed. A trigger event is only produced if both shapes involved have their triggerEvents enabled. A trigger event will produce a ITriggerCallback to the PhysicsShape.callbackTarget for both shapes involved.
