### Description

Get the CompositeCollider2D that is available to be attached to the collider.

Gets any CompositeCollider2D that this Collider2D would use if the Collider geometry is being composited using Collider2D.compositeOperation or returns null if no CompositeCollider2D were available.

Additional resources: CompositeOperation and CompositeCollider2D.
