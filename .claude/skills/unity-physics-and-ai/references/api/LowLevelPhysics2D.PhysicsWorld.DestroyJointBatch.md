### Parameters

| Parameter | Description |
| --- | --- |
| joints | The joints to destroy. |

### Description

Destroy a batch of joints. Any invalid joints will be ignored. Owned joints will produce a warning and will not be destroyed (PhysicsJoint.SetOwner).
