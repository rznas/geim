### Parameters

| Parameter | Description |
| --- | --- |
| input | The configuration of the mover to cast. |

### Returns

**WorldMoverResult** The solved mover results.

### Description

Cast a "Mover" which is geometry designed to collide with the world and solve its movement. Everything is specified via the WorldMoverInput with results returned in WorldMoverResult.
