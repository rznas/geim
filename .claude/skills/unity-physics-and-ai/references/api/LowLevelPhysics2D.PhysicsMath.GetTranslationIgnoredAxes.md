### Parameters

| Parameter | Description |
| --- | --- |
| transformPlane | The transform plane to use. |

### Returns

**Vector3** The ignored translation axes. 0 indicates an axis is used whereas 1 indicates the axis is ignored.

### Description

Get the ignored translation axes, given the specified transform plane. This is the inverse of PhysicsMath.GetTranslationAxes.
