### Description

Set whether to continue or stop the proccesing of collision callbacks if any of the objects involved in the collision are deleted.

During a collision callback, if either a Collider2D or Rigidbody2D that is taking part in that particular callback is deleted then this property controls whether callbacks will continue for this contact or not.

Additional resources: Collider2D, Rigidbody2D.
