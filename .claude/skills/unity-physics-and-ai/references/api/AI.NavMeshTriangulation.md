### Description

Contains data describing a triangulation of a navmesh.

### Properties

| Property | Description |
| --- | --- |
| areas | NavMesh area indices for the navmesh triangulation. |
| indices | Triangle indices for the navmesh triangulation. |
| vertices | Vertices for the navmesh triangulation. |
