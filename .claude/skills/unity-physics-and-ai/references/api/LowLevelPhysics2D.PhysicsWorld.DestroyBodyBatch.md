### Parameters

| Parameter | Description |
| --- | --- |
| bodies | The bodies to destroy. |

### Description

Destroy a batch of bodies. Any invalid bodies will be ignored. Owned bodies will produce a warning and will not be destroyed (See PhysicsBody.SetOwner).
