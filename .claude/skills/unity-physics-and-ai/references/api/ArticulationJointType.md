### Description

The type of the joint that restricts movement of the two connected articulation bodies.

### Properties

| Property | Description |
| --- | --- |
| FixedJoint | Fixed joint doesn't allow any relative movement of the connected bodies. |
| PrismaticJoint | Prismatic joint only allows relative translation of the connection bodies along one specified axis. |
| RevoluteJoint | Revolute joint allows rotational movement around the X axis of the parent's anchor. |
| SphericalJoint | Spherical joint only allows relative rotations of the two connected bodies. |
