### Description

The priority for mixing the PhysicsShape.bounciness properties when two shapes come into contact. If the priority of one shape is higher than the other shape then the higher priority PhysicsShape.SurfaceMaterial.bouncinessMixing will be used. If the priority of both shapes are the same then simply the higher enumeration value of MixingMode from both shapes will be used.
