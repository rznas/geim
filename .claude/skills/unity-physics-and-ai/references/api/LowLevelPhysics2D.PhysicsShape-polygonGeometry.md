### Description

Get/Set the Polygon associated with this shape. When getting the shape geometry, the shape type must match the geometry type otherwise an warning will be produced and an invalid geometry will be returned. Setting the geometry will change the type of shape represented even if the shape type was different before. Setting the geometry will also result in waking the body the shape is attached to.
