### Returns

**NativeArray<LayerHandle>** A NativeArray of all the layer handles added to the composer. This NativeArray must be disposed of after use otherwise leaks will occur. The exception to this is if the array is empty.

### Description

Get the layer handles added to the Physics Composer.
