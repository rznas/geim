# Advanced sculpting tools

**Note:** These tools are only available in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

The advanced sculpting tools automate some complex sculpting tasks. The tools are:

| **Tool** | **Description** |
| --- | --- |
| **Add noise to your terrain** | Adds noise or randomness to the **terrain**. |
| **Build a terrace** | Creates a stepped effect on the terrain. |
| **Clone a part of the terrain** | Copies a section of terrain and pastes it elsewhere. |
| **Build a bridge between two points** | Creates a bridge between two areas of terrain. |
