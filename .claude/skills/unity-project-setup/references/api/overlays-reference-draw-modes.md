# Draw Modes overlay reference

Use the **Draw Modes** overlay to change the way meshes render in the ****Scene**** view. These controls only affect the **Scene** view during development and have no effect on the built game.

| **Draw Mode** | **Icon** | **Description** |
| --- | --- | --- |
| **Wireframe** |  | Draw meshes with a wireframe representation. |
| **Shaded Wireframe** |  | Show meshes textured and with wireframes overlaid. |
| **Unlit** |  | Don’t use scene lighting. |
| **Shaded** |  | Show surfaces with their textures visible. |
| **Debug** |  | Refer to Debug Mode options reference. |
