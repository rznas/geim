### Parameters

| Parameter | Description |
| --- | --- |
| value | The first parameter is the process ID (PID) of the slave process that exited. The second parameter is the ProcessState of the slave process. |

### Description

An event triggered in a master instance of UnityEditor when you start a slave instance with ProcessService.LaunchSlave exit.
