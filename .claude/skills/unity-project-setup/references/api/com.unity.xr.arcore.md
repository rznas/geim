# Google ARCore XR Plugin

com.unity.xr.arcore

## Description

Provides native Google ARCore integration for use with Unity’s multi-platform XR API.
Supports the following features:
-Efficient Background Rendering
-Horizontal Planes
-Depth Data
-Anchors
-Hit Testing
-Occlusion
