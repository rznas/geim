# Downloading and managing Asset Store packages

Once you’ve downloaded or purchased a package from the **Asset Store**, you can use and manage it in the Unity Editor. You can further organize the packages on the Asset Store website.

| **Topic** | **Description** |
| --- | --- |
| **Purchase or download an Asset Store package** | Find packages on the Asset Store, and purchase or download free assets. |
| **Manage packages in the Editor** | Import, update, and manage Asset Store packages in your Unity projects. |
| **Manage packages in the Asset Store** | Organize and find your purchased Asset Store packages on the Asset Store website. |
