### Description

Describes whether a touch is direct, indirect (or remote), or from a stylus.

### Properties

| Property | Description |
| --- | --- |
| Direct | A direct touch on a device. |
| Indirect | An Indirect, or remote, touch on a device. |
| Stylus | A touch from a stylus on a device. |
