# Searching in the Unity Editor

Use the search functionality in the Unity Editor to find and manage resources efficiently in your projects. You can search for assets, **GameObjects**, components, settings, menu commands across different areas of the Editor.

| **Topic** | **Description** |
| --- | --- |
| **Use Classic Search** | Use the classic search in the **Project** and **Hierarchy** windows and in the ****Scene**** view. |
| **Unity Search** | Find and manage assets, GameObjects, menu items, settings, and other resources in your project with Unity Search. |
