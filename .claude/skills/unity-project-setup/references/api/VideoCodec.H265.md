### Description

Encode video with the H.265 codec.

The MPEG-H Part 2, or High Efficiency Video Coding (HEVC) video codec. This codec is supported by hardware on most platforms. For more information about video encoding, refer to Video encoding compatibility reference. 

**Note**: The codec is not included with Windows by default. You might need to manually install the necessary codec on your Windows device to ensure compatibility.
