### Description

The Wind module implements the WindZone component which can affect terrain rendering and particle simulations.

### Classes

| Class | Description |
| --- | --- |
| WindZone | Wind Zones add realism to the trees you create by making them wave their branches and leaves as if blown by the wind. |

### Enumerations

| Enumeration | Description |
| --- | --- |
| WindZoneMode | Modes a Wind Zone can have, either Spherical or Directional. |
