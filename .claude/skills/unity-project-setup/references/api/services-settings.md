# Services

Use the **Services** settings to view and apply settings for using your project in the Unity Dashboard. To access the Services settings, go to **Edit** > **Project Settings** then select the **Services** category.

| **Property** | **Description** |
| --- | --- |
| **Project Name** | Displays the project name. You can edit the project name in the Unity Dashboard. |
| **Unity Organization** | Displays the organization linked to the project. |
| **Unity Project ID** | Displays the ID of the project. This is used to connect with the Unity Dashboard. To link to a different project in the Dashboard, select **Unlink project** and then follow the steps in [Configure a project for Unity Cloud](https://docs.unity.com/cloud/en-us/projects/configure-project-for-unity-cloud). |
| **Members** | Define who can view your project in the Unity Dashboard. For more information, refer to [Visibility of Projects](https://docs.unity.com/cloud/en-us/projects#visibility-of-projects). |
| **Will this app be targeted to children as defined by applicable laws?** | Specify whether or not the app is intended for children. For more information, refer to [Child data law compliance](https://docs.unity.com/grow/en-us/ads/privacy/coppa-compliance). |
