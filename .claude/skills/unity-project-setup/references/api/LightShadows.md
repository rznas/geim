### Description

Shadow casting options for a Light.

Additional resources: light component.

### Properties

| Property | Description |
| --- | --- |
| None | Do not cast shadows (default). |
| Hard | Cast "hard" shadows (with no shadow filtering). |
| Soft | Cast "soft" shadows (with 4x PCF filtering). |
