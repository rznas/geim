### Description

Defines the types of text generators to use.

This enum is used to switch between Unity's standard and advanced text generators.

### Properties

| Property | Description |
| --- | --- |
| Standard | The standard text generator, which is the default option. |
| Advanced | Supports comprehensive Unicode and text shaping for various languages and scripts, including right-to-left (RTL) languages. |
