### Description

Swizzles cell positions to other positions.

### Properties

| Property | Description |
| --- | --- |
| XYZ | Keeps the cell positions at XYZ. |
| XZY | Swizzles the cell positions from XYZ to XZY. |
| YXZ | Swizzles the cell positions from XYZ to YXZ. |
| YZX | Swizzles the cell positions from XYZ to YZX. |
| ZXY | Swizzles the cell positions from XYZ to ZXY. |
| ZYX | Swizzles the cell positions from XYZ to ZYX. |
