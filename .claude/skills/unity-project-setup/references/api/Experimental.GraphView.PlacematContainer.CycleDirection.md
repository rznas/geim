### Description

Use to set the placemat cycling direction.

### Properties

| Property | Description |
| --- | --- |
| Up | Set the placemat cycle direction to up. Use when cycling a placemat up. |
| Down | Set the placemat cycle direction to down. Use when cycling a placemat down. |
