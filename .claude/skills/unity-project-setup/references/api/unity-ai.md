# Unity AI

Use Unity AI to accelerate development, automate workflows, and generate assets with artificial intelligence (AI) tools integrated into the Unity Editor and Unity Dashboard.

Unity provides the following AI-powered tools to support game development workflows in the Unity Editor and at runtime:

- **Assistant**: provides in-Editor guidance to write code, troubleshoot, and automate tasks.
- **Generators**: creates assets, such as **sprites**, textures, sounds, animations, and materials, based on text prompts or reference inputs.
- **Sentis**: enables you to integrate and run trained machine learning (ML) models on end-user devices or directly in the Unity Editor.

You can also use the [Unity Dashboard](https://cloud.unity.com/home/ai/settings) - a cloud-based interface that provides a centralized view of your AI usage, settings, and point tracking. It helps you monitor activity, manage configurations, and control access to Unity AI features across your projects. For more information, refer to the [Unity Dashboard AI settings](https://docs.unity.com/en-us/ai).
