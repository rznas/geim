### Description

A structure that contains the connection information of a Channel in ChannelService.

### Properties

| Property | Description |
| --- | --- |
| id | The ID of a specific channel. Normally this is the hash of the channel name. |
| name | The name of the channel. This matches the route of the URL that connects to the ChannelService. For example, "127.0.0.1:9292/<channelName>". |
