### Description

Interface into functionality unique to handheld devices.

This class is available on Android and iOS.

### Static Methods

| Method | Description |
| --- | --- |
| GetActivityIndicatorStyle | Gets the current activity indicator style. |
| PlayFullScreenMovie | Plays a full-screen movie. |
| SetActivityIndicatorStyle | Sets the desired activity indicator style. |
| StartActivityIndicator | Starts os activity indicator. |
| StopActivityIndicator | Stops os activity indicator. |
| Vibrate | Triggers device vibration. |
