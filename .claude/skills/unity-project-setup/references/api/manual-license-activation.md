# Manual license activation

If you can’t activate your license via the Unity Hub or the command line, you can activate your license manually if you have a serial key.

| **Topic** | **Description** |
| --- | --- |
| **Introduction to manual license activation** | Understand the manual license activation process if you can’t activate via the Hub or command line. |
| **Submit a license request from the Hub** | Generate a license request via the Hub and upload it using a browser. |
| **Submit a license request from a command line and browser (Windows)** | Generate a request from the Windows command prompt, upload it online, and install the license. |
| **Submit a license request from a command line and browser (macOS, Linux)** | Generate a request from the macOS or Linux terminal, upload it online, and install the license. |
