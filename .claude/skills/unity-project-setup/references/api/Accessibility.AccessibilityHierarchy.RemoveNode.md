### Parameters

| Parameter | Description |
| --- | --- |
| node | The node to remove. |
| removeChildren | `true` if the node's descendants should also be removed, or `false` if they should be moved under the node's parent. Defaults to `true`. |

### Description

Removes the node from the accessibility hierarchy and removes or re-parents its descendants.
