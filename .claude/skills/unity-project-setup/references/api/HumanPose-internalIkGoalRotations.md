### Description

The rotations of the four IK goals: left foot, right foot, left hand and right hand, as used internally in the Humanoid system.

These values can be used to create Avatar IK goal rotation keyframes in Humanoid AnimationClips.

 **Note**: HumanPose.internalIkGoalRotations cannot be used directly to calculate global rotations, as they are offset to follow internal Humanoid logic. To calculate global IK goal rotations, see HumanPose.ikGoalRotations.
