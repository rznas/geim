### Description

Class containing methods to interact with the selected Unity PlayModeView (GameView, Simulator).

### Static Methods

| Method | Description |
| --- | --- |
| GetPlayModeFocused | Returns the enter play mode behavior for the selected PlayModeView. |
| GetRenderingResolution | Retrieves the current rendering resolution of the selected PlayModeView. |
| GetViewType | Gets the type of the selected PlayModeView window. |
| SetCustomRenderingResolution | Adds a new Custom resolution entry with the specified baseName. If an entry with the same baseName already exists, this method updates the resolution of this entry with the new values. |
| SetPlayModeFocused | Changes the enter play mode behavior of the selected PlayModeView window. |
| SetViewType | Changes the type of the selected PlayModeView window. |
