# Submit a UPM package for approval

**Note**: Paid **UPM packages** are not yet available on the Publisher Portal. You can publish free UPM packages.

The **Asset Store** team reviews each package draft to make sure it meets the [Submission Guidelines](https://unity3d.com/asset-store/sell-assets/submission-guidelines) before you can publish it to the Asset Store.

## Submit a UPM package to the Publisher Portal

1. Open the Publisher Portal and select **Products**.
2. In the **Package** tab, choose your package configuration and set a namespace. To learn more, refer to Create a Publisher Portal draft.
3. In the **Product information** tab, add detailed information about the product, and any tags.
4. In the **Media** tab, add videos, marketing images, and screenshots of the product.
5. Select **Submit** to submit the package for approval. Choose from the following options:
  - **Publish immediately**: Your product goes live as soon as Unity reviews and approves it.
  - **Manually publish**: Get notified as soon as Unity reviews and approves your product, and then you can choose when to publish.

The Asset Store team takes at least five business days to analyze your submission. If you don’t receive a response after two weeks, you can contact [the Unity support team](https://support.unity3d.com/hc/en-us/requests/new?ticket_form_id=65905&_ga=2.71465235.1219756845.1585343727-1395087459.1585343727) to ask for an update.

## Manage a rejected package submission

If Unity rejects your package submission, you can review the reason for rejection in the Publisher Portal. If the portal displays the message **Rejection reasons sent by email**, check your email for information about the reason for rejection.
