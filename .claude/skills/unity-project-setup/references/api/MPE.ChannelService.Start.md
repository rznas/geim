### Description

Starts the ChannelService. After you start the ChannelService it listens to connection at the URL provided by: <ChannelService Address>:<ChannelServicePort>/<channelName>, for example, 127.0.0.1:9976/events. See ChannelService.GetAddress and ChannnelService.GetPort.
