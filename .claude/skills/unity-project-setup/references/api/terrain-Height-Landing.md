# Height

The **Terrain**’s **Paint Terrain** tools sculpt the terrain to create mountains, valleys, and other topographical features. The primary height tools are available in both the built-in Terrain system and the Terrain Tools package. The advanced height tools and a selection of effect, transform, and erosion tools are available only in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

**Note:** For information about the **Paint Texture** tool, refer to Textures and layers.

| **Topic** | **Description** |
| --- | --- |
| **Before you start** | Factors to consider before you start sculpting. |
| **Primary sculpting tools** | Sculpt your world’s main features. |
| **Advanced sculpting tools** | Add noise, build terraces and bridges, or clone parts of the terrain. These tools are only available in the Terrain Tools package. |
| **Change existing height with effects** | Flatten or sharpen the terrain or add contrast between heights. These tools are only available in the Terrain Tools package. |
| **Transform terrain features** | Pinch, smudge or twist the terrain. These tools are only available in the Terrain Tools package. |
| **Erode the terrain** | Erode the terrain with wind, thermal, and hydraulic tools. These tools are only available in the Terrain Tools package. |
