# Unity Editor settings reference

Use the Unity Editor Preferences and **Project Settings** to configure how the Editor and your project behaves.

Preferences are personal settings stored on your local machine. Project Settings are shared settings that define how your project builds and runs, and are typically stored in **version control** for your entire team.

| **Topic** | **Description** |
| --- | --- |
| **Preferences reference** | Configure personal Editor preferences that apply to your computer across all projects. |
| **Project Settings reference** | Manage project-specific settings that control how your application builds, renders, and behaves. |
