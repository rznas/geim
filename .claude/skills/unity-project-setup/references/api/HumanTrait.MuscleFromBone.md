### Parameters

| Parameter | Description |
| --- | --- |
| i | Bone index. |
| dofIndex | Number representing a "degree of freedom": 0 for X-Axis, 1 for Y-Axis, 2 for Z-Axis. |

### Description

Obtain the muscle index for a particular bone index and "degree of freedom".

The indexing order of the bones is the same as that of the BoneName array.

Additional resources: HumanTrait.BoneName, HumanTrait.BoneCount, HumanTrait.MuscleName, HumanTrait.MuscleCount.
