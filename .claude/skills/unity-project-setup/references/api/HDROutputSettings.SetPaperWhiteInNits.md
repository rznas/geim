### Description

Sets the base luminance of a white paper surface in nits or candela per square meter (cd/m2).

The base luminance of a white paper surface in nits or candela per square meter (cd/m2). Unity uses this value if HDROutputSettings.automaticHDRTonemapping is true. The higher this value, the brighter the resulting image appears on the HDR display.

This method has been deprecated. Please use HDROutputSettings.paperWhiteNits instead.
