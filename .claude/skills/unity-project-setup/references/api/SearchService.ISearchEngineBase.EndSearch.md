### Description

A function called at the end of a search.

A search is performed every time the search query is modified. This function is not called for IObjectSelectorEngines.

Additional resources: ISearchEngineBase.BeginSearch, ISearchEngineBase.BeginSession, ISearchEngineBase.EndSession.
