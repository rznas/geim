### Description

A selection of modes that control when Unity executes the 2D physics simulation.

Additional resources: Physics2D.simulationMode.

### Properties

| Property | Description |
| --- | --- |
| FixedUpdate | Use this enumeration to specify to Unity that it should execute the physics simulation immediately after the MonoBehaviour.FixedUpdate. |
| Update | Use this enumeration to specify to Unity that it should execute the physics simulation immediately after MonoBehaviour.Update. |
| Script | Use this enumeration to specify to Unity that it should execute the physics simulation manually when you call Physics2D.Simulate. |
