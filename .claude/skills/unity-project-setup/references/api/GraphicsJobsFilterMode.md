### Description

Enumerates Graphics Jobs modes to use when device filtering.

### Properties

| Property | Description |
| --- | --- |
| Off | Disables Graphics Jobs. |
| Native | Native Graphics Jobs mode. |
| Legacy | Legacy Graphics Jobs mode. |
| Split | Split Graphics Jobs mode. |
