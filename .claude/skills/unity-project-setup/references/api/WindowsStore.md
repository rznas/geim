# Universal Windows Platform

Universal Windows Platform (UWP) provides a common app platform on every device that runs Windows, such as Windows 10+, and more. You can use UWP to build your Unity apps for any Windows device and publish them to the Microsoft Store, without needing to rewrite the app for each platform.
