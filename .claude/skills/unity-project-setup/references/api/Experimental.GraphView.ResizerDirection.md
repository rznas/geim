### Description

Enum that specifies in which direction to resize the element.

### Properties

| Property | Description |
| --- | --- |
| Top | Resize the element from the top. |
| Bottom | Resize the element from the bottom. |
| Left | Resize the element to the left. |
| Right | Resize the element to the right. |
