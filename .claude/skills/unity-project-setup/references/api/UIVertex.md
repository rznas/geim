### Description

Vertex class used by a Canvas for managing vertices.

### Static Properties

| Property | Description |
| --- | --- |
| simpleVert | Simple UIVertex with sensible settings for use in the UI system. |

### Properties

| Property | Description |
| --- | --- |
| color | Vertex color. |
| normal | Normal. |
| position | Vertex position. |
| tangent | Tangent. |
| uv0 | The first texture coordinate set of the mesh. Used by UI elements by default. |
| uv1 | The second texture coordinate set of the mesh, if present. |
| uv2 | The Third texture coordinate set of the mesh, if present. |
| uv3 | The forth texture coordinate set of the mesh, if present. |
