### Description

A readonly struct describing a contact point between two Colliders.

### Properties

| Property | Description |
| --- | --- |
| impulse | The impulse applied to this contact pair to resolve the collision. |
| normal | Normal of the contact point. |
| position | The position of the contact point between the Colliders, in world space. |
| separation | The distance between the edges of Colliders at the contact point. |
