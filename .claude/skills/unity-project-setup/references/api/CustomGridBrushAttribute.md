### Description

Attribute to define the class as a grid brush and to make it available in the palette window.

### Properties

| Property | Description |
| --- | --- |
| defaultBrush | If set to true, brush will replace Unity built-in brush as the default brush in palette window.Only one class at any one time should set defaultBrush to true. |
| defaultName | Name of the default instance of this brush. |
| hideAssetInstances | Hide all asset instances of this brush in the tile palette window. |
| hideDefaultInstance | Hide the default instance of brush in the tile palette window. |

### Constructors

| Constructor | Description |
| --- | --- |
| CustomGridBrushAttribute | Attribute to define the class as a grid brush and to make it available in the palette window. |
