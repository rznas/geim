### Description

The bounding box mode for generating a grid of interpolated Light Probes.

Additional resources: Light Probes, Light Probe Proxy Volume.

### Properties

| Property | Description |
| --- | --- |
| AutomaticLocal | The bounding box encloses the current Renderer and all the relevant Renderers down the hierarchy, in local space. |
| AutomaticWorld | The bounding box encloses the current Renderer and all the relevant Renderers down the hierarchy, in world space. |
| Custom | A custom local-space bounding box is used. The user is able to edit the bounding box. |
