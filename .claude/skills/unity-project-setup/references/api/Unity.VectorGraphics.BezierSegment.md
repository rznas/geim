### Description

A bezier segment.

Cubic Bezier segment starts from `P0`, flies in tangent to direction from `P0` to `P1`, then lands in direction from `P2` to `P3`, to finally end exactly at `P3`. 

 For more information, refer to work with vector graphics.

### Properties

| Property | Description |
| --- | --- |
| P0 | Origin point of the segment. |
| P1 | First control point of the segment. |
| P2 | Second control point of the segment. |
| P3 | Ending point of the segment. |
