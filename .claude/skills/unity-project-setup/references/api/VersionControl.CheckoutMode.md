### Description

What to checkout when starting the Checkout task through the version control Provider.

### Properties

| Property | Description |
| --- | --- |
| Asset | Checkout the asset only. |
| Meta | Checkout .meta file only. |
| Both | Checkout both asset and .meta file. |
| Exact | Checkout only those assets provided in the asset list. |
