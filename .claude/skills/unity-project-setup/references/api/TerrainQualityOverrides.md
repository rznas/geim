### Description

Flags used by QualitySettings to specify which Terrain fields the quality settings should override.

### Properties

| Property | Description |
| --- | --- |
| None | Override no field. |
| PixelError | Override Terrain.heightmapPixelError. |
| BasemapDistance | Override Terrain.basemapDistance. |
| DetailDensity | Override Terrain.detailObjectDensity. |
| DetailDistance | Override Terrain.detailObjectDistance. |
| TreeDistance | Override Terrain.treeDistance. |
| BillboardStart | Override Terrain.treeBillboardDistance. |
| FadeLength | Override Terrain.treeCrossFadeLength. |
| MaxTrees | Override Terrain.treeMaximumFullLODCount. |
