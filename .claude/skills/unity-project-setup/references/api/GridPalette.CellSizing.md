### Description

Controls the sizing of cells for a Palette.

### Properties

| Property | Description |
| --- | --- |
| Automatic | Automatically resizes the Palette cells by the size of Sprites in the Palette. |
| Manual | Size of Palette cells will be changed manually by the user. |
