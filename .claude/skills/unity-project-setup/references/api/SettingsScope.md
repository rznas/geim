### Description

Sets the scope of a SettingsProvider. The Scope determines where it appears in the UI. For example, whether it appears with the Project settings in the Settings window, or in the Preferences window, or in both windows.

Additional resources: SettingsProvider.scope, SettingsProvider.SettingsProvider.

### Properties

| Property | Description |
| --- | --- |
| User | The SettingsProvider appears only in the Preferences window. |
| Project | The SettingsProvider appears only in the Project Settings window. |
