### Description

The status of the on-screen keyboard.

### Properties

| Property | Description |
| --- | --- |
| Visible | The on-screen keyboard is visible. |
| Done | The user has finished providing input. |
| Canceled | The on-screen keyboard was canceled. |
| LostFocus | The on-screen keyboard has lost focus. |
