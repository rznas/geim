# Unity Version Control

com.unity.collab-proxy

## Description

The package gives you the ability to use Unity Version Control in the Unity editor. To use Unity Version Control, a subscription is required. Learn more about how you can get started for free by visiting [https://unity.com/solutions/version-control](https://unity.com/solutions/version-control)
