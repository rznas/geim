### Description

Enumeration for SystemInfo.operatingSystemFamily.

Determines the operating system family application is running on.

### Properties

| Property | Description |
| --- | --- |
| Other | Returned for operating systems that do not fall into any other category. |
| MacOSX | macOS operating system family. |
| Windows | Windows operating system family. |
| Linux | Linux operating system family. |
