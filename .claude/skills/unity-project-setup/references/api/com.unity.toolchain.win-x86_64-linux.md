# Toolchain Windows x64 Host for Linux Targets

com.unity.toolchain.win-x86_64-linux

## Description

This package provides a cross-compilation toolchain for building Unity players targeting Linux and Embedded Linux platforms. It is designed to run on 64-bit(x86_64) Windows hosts and supports both x86_64 and Arm64 player targets
