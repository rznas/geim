# Overlays preferences reference

Customize these settings to set the background color of overlays for each Editor window that supports them.

To open these preferences, go to **Edit** > **Preferences** > **Overlays** (macOS: **Unity** > **Settings** > **Overlays**).

## Scene View

The ****Scene** View** section contains the following settings.

| **Setting** | **Description** |
| --- | --- |
| **Use Default** | Resets the **Overlay Background** color for that window to the default for the active **Editor Theme**. |
| **Overlay Background** | Sets the background color of floating overlays in the **Scene view**. The default is a translucent dark gray for the **Dark** theme, and a translucent light gray for the **Light** theme. |
