### Description

The maximum number of bones affecting a single vertex.

Additional resources: SkinnedMeshRenderer.quality, QualitySettings.skinWeights.

### Properties

| Property | Description |
| --- | --- |
| Auto | Chooses the number of bones from the number current QualitySettings. (Default) |
| Bone1 | Use only 1 bone to deform a single vertex. (The most important bone will be used). |
| Bone2 | Use 2 bones to deform a single vertex. (The most important bones will be used). |
| Bone4 | Use 4 bones to deform a single vertex. |
