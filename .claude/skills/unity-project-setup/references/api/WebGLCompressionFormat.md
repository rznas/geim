### Description

An enum containing different compression types.

This enum is used within the WebGL platform to define what type of compression will be used for resources.

### Properties

| Property | Description |
| --- | --- |
| Brotli | WebGL resources are stored using Brotli compression. |
| Gzip | WebGL resources are stored using Gzip compression. |
| Disabled | WebGL resources are uncompressed. |
