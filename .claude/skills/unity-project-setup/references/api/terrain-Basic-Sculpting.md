# Primary sculpting tools

Primary tools for sculpting height are available in both the built-in **Terrain** system and the Terrain Tools package. The tools are:

| **Tool** | **Description** |
| --- | --- |
| **Set the height of an area or tile** | Set a single height for a whole tile or parts of it. |
| **Sculpt the terrain with a brush** | Sculpt areas of the terrain. |
| **Paint holes in the terrains** | Paint holes in the terrain to allow characters and objects to enter caves or other underground structures. |
| **Smooth the terrain’s height** | Smooth sharp formations and transitions between areas of the terrain. |
| **Stamp a shape onto the terrain** | Use a stamp or **mesh** to add or remove height as a set shape. |
