### Description

Represent the connection state of the version control provider.

### Properties

| Property | Description |
| --- | --- |
| Updating | The version control provider is currently trying to connect to the version control server. |
| Online | Connection to the version control server has been established. |
| Offline | Connection to the version control server could not be established. |
