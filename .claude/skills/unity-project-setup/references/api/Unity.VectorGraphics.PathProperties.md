### Description

Defines properties of paths.

### Properties

| Property | Description |
| --- | --- |
| Corners | How the corners of the path are displayed. |
| Head | How the beginning of the path is displayed. |
| Stroke | The stroke used to render the path. |
| Tail | How the end of the path is displayed. |
