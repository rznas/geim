### Parameters

| Parameter | Description |
| --- | --- |
| node | The hierarchy node. |
| sortIndex | The sort index. |

### Description

Sets the sort index of a hierarchy node.

After setting sort indexes, you must call SortChildren or SortChildrenRecursive on the parent node to sort the child nodes.
