### Description

Indicates a change to the Terrain holes data.

This flag is set when there are changes to the Terrain holes data, for example, after calls to TerrainData.SetHoles. Note that calls to TerrainData.SetHolesDelayLOD don't set this flag, even though those calls update the Terrain holes data. See DelayedHolesUpdate.
