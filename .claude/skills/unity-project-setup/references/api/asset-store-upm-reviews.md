# Check and manage UPM product reviews

Any user that has downloaded your product can leave a public review on your product description page. A review includes a star rating between one and five and a feedback message. To manage reviews for your products in the Publisher Portal, select the **Reviews** section. Unread reviews appear in bold text. Select the filter dropdowns to sort by rating or product. Select the Menu (**…**) icon to sort the listed reviews and manage visibility.

## Reply to a review

To send a reply to a user review:

1. Select a review.
2. In the review panel that appears, enter a reply in the **Your reply** field.
3. Select **Reply**.

**Note**: For guidance on how to respond to positive and negative feedback, select the **Guidance** tab in the review panel.

## Report a review

You can report a review for the following content:

- Offensive language.
- Discussing topics irrelevant to the package.
- Violation of any other prohibitions of [Unity’s Website and Communities Terms](https://unity.com/legal/terms-of-service/site-and-communities).

To report a review:

1. Go to the product description page for your product.
2. Select the **Reviews** tab.
3. Find the review you want to report.
4. Select **Report this review**.
