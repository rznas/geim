### Description

Mac fullscreen mode.

This defines how fullscreen mode is handled on macOS Standalones.

### Properties

| Property | Description |
| --- | --- |
| FullscreenWindow | Fullscreen window. |
| FullscreenWindowWithDockAndMenuBar | Fullscreen window with Dock and Menu bar. |
