### Description

Sets the contact filter to filter within the minDepth and maxDepth range, or outside that range.

useOutsideDepth uses the range between minDepth and maxDepth to filter depth. Set `useOutsideDepth` to true to only return object depth outside that range. Set `useOutsideDepth` to false to only return object depth within that range. It is false by default.

Additional resources: minDepth, maxDepth & useDepth.
