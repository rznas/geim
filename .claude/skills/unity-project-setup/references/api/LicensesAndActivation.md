# Licenses and activation

To use Unity, you need an activated license. To learn about Unity’s plans and prices, go to the [Unity Store](https://store.unity.com/).

| License troubleshooting | Troubleshoot your license management issues. |

## Additional resources

- If you have a complex project or a large organization, you might consider Unity’s on-premises floating licensing server solution, rather than standard user licenses. For information, refer to Unity Licensing Server.
- Unity Support article, [Licenses](https://support.unity.com/hc/categories/201268913-Licenses).
