### Description

Utility class to generate search column with item selectors.

### Static Methods

| Method | Description |
| --- | --- |
| CreateColumn | Create a search column. |
| Enumerate | Enumerate search column for a given set of search items. |
| GetItemContentStyle | Returns the default item GUI style used to draw the cell of a search item property in table view. |
