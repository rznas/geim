# Visual Effect Graph asset

A Visual Effect Graph asset contains all of the necessary data, such as the graph and **shaders**, that Unity requires to process an effect

This Asset only works when you install the Visual Effect Graph package. For information on how to install the Visual Effect Graph, refer to Install a package from a registry.

For information on how to use Visual Effect Graph Assets, refer to the Visual Effect Graph Assets package documentation.
