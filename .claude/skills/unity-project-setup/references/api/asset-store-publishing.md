# Publishing Asset Store packages

You can publish packages to the **Asset Store** to share and monetize assets, tools, and extensions. You can publish packages in the **asset package** (`.unitypackage` files) format, or apply to publish in the Unity Package Manager (UPM) beta program.

| **Topic** | **Description** |
| --- | --- |
| **Introduction to publishing Asset Store packages** | Understand the differences between publishing asset packages and Unity Package Manager packages on the Asset Store. |
| **Set up an Asset Store publisher profile** | Create a Unity ID and a publisher profile that you use to manage and publish Asset Store packages. |
| **Add a user to a publisher profile** | Allow access to your publisher portal account from multiple email addresses. |
| **Publish an asset package on the Asset Store** | Publish a package to the Asset Store in the asset package (`.unitypackage`) format. |
| **Publish a UPM package on the Asset Store** | Publish a package to the Asset Store in the Unity Package Manager (UPM) format. |
