# Issue an Asset Store voucher

Use a voucher to give a customer one of your paid packages for free. You can create up to 16 vouchers a year for each **Asset Store** package.

A customer can redeem a voucher in the Asset Store cart or the [redeem a voucher](https://assetstore.unity.com/account/voucher) page.

**Note:** You can only give vouchers away for free. You can’t sell or exchange them.

To issue a voucher:

1. Open the [Asset Store Publisher](https://publisher.assetstore.unity3d.com/) portal.
2. Select the **Vouchers** tab.
3. Select the **Create new voucher** button.
4. Enter the package name, number of vouchers and type of voucher code you want to create.
5. Select **Create**.
