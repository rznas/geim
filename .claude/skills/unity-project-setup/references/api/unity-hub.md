# Install Unity using the Hub

The Unity Hub is the best way to install the Unity Editor, manage your project, and access learning resources.

To download the Hub, go to [Download Unity](https://unity.com/download). To learn how to install the Hub on different operating systems, refer to [Install the Unity Hub](https://docs.unity.com/hub/install-hub).
