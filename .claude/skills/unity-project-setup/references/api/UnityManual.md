# Unity 6.4 User Manual

Unity 6.4 is the latest release of the next generation of the Unity Engine. It combines the latest technologies and tools to deliver high-quality, high-performance experiences for all supported platforms. Refer to the Unity download page to [download the latest version of Unity](https://unity.com/download).

For more information about new features and how to upgrade your existing projects to this version, refer to the following topics:

- New in Unity
- Upgrade Unity

## Highlights of Unity 6

Explore the key features and updates introduced in the Unity 6 family of releases.

### Boost rendering performance

Elevate your **scenes** with scalable, captivating visuals using the latest advances in rendering, lighting, and visual effects.

### Multiplayer game creation

Simplify multiplayer game creation with Unity’s multiplayer packages and services.

### Expand multiplatform reach

Build better experiences for mobile platforms, including a newly optimized runtime for mobile browsers, and get the latest multiplatform advances for all supported platforms.

### Unlock possibilities with Unity AI

Accelerate creativity and development with AI tools for [code generation](https://docs.unity3d.com/Packages/com.unity.ai.assistant@latest), [asset creation](https://docs.unity3d.com/Packages/com.unity.ai.generators@latest), and [runtime inference](https://docs.unity3d.com/Packages/com.unity.ai.inference@latest), all integrated into the Unity Editor and Dashboard.

### Achieve more engaging visuals

Create more engaging visuals with the latest updates to Lighting, Graphics performance and profiling, [Shader Graph](https://docs.unity3d.com/Packages/com.unity.shadergraph@latest/), and [Visual Effect Graph](https://docs.unity3d.com/Packages/com.unity.visualeffectgraph@latest/).

### [Enhance productivity and functionality](https://unity.com/features/cinemachine)

Improve productivity and functionality across your entire Unity development environment with better profiling options, [ProBuilder](https://docs.unity3d.com/Packages/com.unity.probuilder@latest/), [Cinemachine](https://docs.unity3d.com/Packages/com.unity.cinemachine@latest/), and UI Toolkit.
