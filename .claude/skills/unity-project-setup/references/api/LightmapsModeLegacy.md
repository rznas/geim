Additional resources: LightmapSettings.lightmapsModeLegacy.

### Properties

| Property | Description |
| --- | --- |
| Single | Single, traditional lightmap rendering mode. |
| Dual | Dual lightmap rendering mode. |
| Directional | Directional rendering mode. |
