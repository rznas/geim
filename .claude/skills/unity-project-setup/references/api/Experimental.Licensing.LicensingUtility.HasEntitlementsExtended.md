### Parameters

| Parameter | Description |
| --- | --- |
| entitlements | The list of entitlements for which we want extended information. |
| includeCustomData | If we wish to include custom data in the response |

### Returns

**void** Returns a list of EntitlementInfo data structures for each of the requested entitlements, if available.

### Description

Retrieves extended information about the license file and its contents, including custom data attached to individual entitlements.
