### Description

Wrapping modes for text that reaches the horizontal boundary.

### Properties

| Property | Description |
| --- | --- |
| Wrap | Text will word-wrap when reaching the horizontal boundary. |
| Overflow | Text can exceed the horizontal boundary. |
