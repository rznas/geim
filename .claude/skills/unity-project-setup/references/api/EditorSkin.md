### Description

Enum that selects which skin to return from EditorGUIUtility.GetBuiltinSkin.

### Properties

| Property | Description |
| --- | --- |
| Game | The skin used for game views. |
| Inspector | The skin used for inspectors. |
| Scene | The skin used for Scene views. |
