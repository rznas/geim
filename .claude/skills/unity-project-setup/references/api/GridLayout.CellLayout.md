### Description

The layout of the GridLayout.

The layout determines the conversion of positions from cell space to local space and vice versa.

### Properties

| Property | Description |
| --- | --- |
| Rectangle | Rectangular layout for cells in the GridLayout. |
| Hexagon | Hexagonal layout for cells in the GridLayout. |
| Isometric | Isometric layout for cells in the GridLayout. |
| IsometricZAsY | Isometric layout for cells in the GridLayout where any Z cell value set will be added as a Y value. |
