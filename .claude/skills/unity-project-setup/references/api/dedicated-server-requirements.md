# Install Dedicated Server module

To build for the Dedicated Server platform, you must install the Dedicated Server module. Before installing the module, ensure your system meets the requirements for the server platform.

1. Download and install the Unity Editor through Unity Hub. Dedicated Server build support is available from Unity Editor version 2021.3 LTS and later. During the installation of the Unity Editor, you can add modules to extend the platform build support.  **Note**: If you’ve already installed the Editor, refer to Add modules to install the required module.
2. Enable the Dedicated Server build support for the required platform you want to build your application on. For example, enable **Linux Dedicated Server Build Support** if you want to build for Linux.
3. Select **Install**.
