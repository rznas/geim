### Description

Defines the type of line endings used when creating new C# files from within the editor.

### Properties

| Property | Description |
| --- | --- |
| OSNative | File endings are the default for the active platform. On Windows, this is a carriage return character followed by linefeed character (\r\n). On macOS or Linux, this is a single linefeed characater (\n). |
| Unix | Lines are terminated with a newline character. Also referred to as linefeed, or LF. |
| Windows | Lines are terminated with carriage return and newline characters. Also referred to as CR/LF or `\r\n`. |
