# Manage search queries

Search queries are the terms and expressions you use to find specific assets, settings, and menu items. You can save frequent queries to reuse them later or invoke them from custom menu items to speed up your workflow.

| **Topic** | **Description** |
| --- | --- |
| **Save and reuse search queries** | Save search queries or share them across the project. |
| **Invoke from the main menu** | Create a custom menu item that invokes a specific search. |
