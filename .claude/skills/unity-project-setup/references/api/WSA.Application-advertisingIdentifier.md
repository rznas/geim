### Description

Advertising ID.

Windows::System::UserProfile::AdvertisingManager::AdvertisingId. **Note: If you get an empty string, check 'PC Settings -> Privacy -> Let apps use my advertising ID for experiences across apps (turning this off will reset your ID)' option.**
