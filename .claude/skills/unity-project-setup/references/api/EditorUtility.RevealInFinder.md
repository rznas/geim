### Parameters

| Parameter | Description |
| --- | --- |
| path | The path of the file to reveal. This parameter is relative to the project directory. For example, "Assets/foo.txt" highlights the Assets directory when Finder opens. |

### Description

Opens a Finder window (macOS) or File Explorer window (Microsoft Windows) and points it to the specified file.
