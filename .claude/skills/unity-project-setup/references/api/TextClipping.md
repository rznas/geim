### Description

Different methods for how the GUI system handles text being too large to fit the rectangle allocated.

### Properties

| Property | Description |
| --- | --- |
| Overflow | Text flows freely outside the element. |
| Clip | Text gets clipped to be inside the element. |
| Ellipsis | Text gets clipped to be inside the element and added ... at the end. |
