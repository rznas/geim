### Parameters

| Parameter | Description |
| --- | --- |
| id | The ID of the node to retrieve. |
| node | The valid node with the associated ID, or `null` if no such node exists in this hierarchy. |

### Returns

**bool** `true` if a node is found and `false` otherwise.

### Description

Tries to retrieve the AccessibilityNode with the given ID in the hierarchy.
