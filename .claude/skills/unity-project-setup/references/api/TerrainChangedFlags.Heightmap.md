### Description

Indicates a change to the heightmap data.

This flag is set when heightmap data has been changed, for instance after TerrainData.SetHeights calls. Note that TerrainData.SetHeightsDelayLOD doesn't set this flag, even though the heightmap data is updated. See DelayedHeightmapUpdate.
