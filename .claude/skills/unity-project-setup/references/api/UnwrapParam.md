### Description

Unwrapping settings.

Allows you control over the parameters of the automatic UV Unwrapping feature. Use this if you want to override the default unwrap settings.

### Properties

| Property | Description |
| --- | --- |
| angleError | Maximum allowed angle distortion (0..1). |
| areaError | Maximum allowed area distortion (0..1). |
| hardAngle | This angle (in degrees) or greater between triangles will cause seam to be created. |
| packMargin | How much uv-islands will be padded. |

### Static Methods

| Method | Description |
| --- | --- |
| SetDefaults | Will set default values for params. |
