### Description

An unique identifier for the issue descriptor (read-only).

Reports can contain two different types of ReportItem: - Issues, which indicate a potential problem which should be investigated and possibly fixed: for example, a texture with its Read/Write Enabled checkbox ticked. - Insights, for informational purposes: for example, general information about a texture in the project.

 Issues can be identified by having a valid DescriptorId. See also: the IsIssue method.
