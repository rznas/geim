### Description

Defines the axes that can be snapped.

### Properties

| Property | Description |
| --- | --- |
| None | No axes support snapping. |
| X | Snapping is available only on the \x\ axis. |
| Y | Snapping is available only on the \y\ axis. |
| Z | Snapping is available only on the \z\ axis. |
| All | Snapping is available on all axes: x, y, and z. |
