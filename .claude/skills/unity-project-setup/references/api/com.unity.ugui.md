# uGUI

com.unity.ugui

## Description

uGUI (Unity UI) is a set of tools for developing user interfaces for games and applications. It is a GameObject-based UI system that uses Components and the Game View to arrange, position, and style user interfaces. ​ You cannot use Unity UI to create or change user interfaces in the Unity Editor.

## Version information

Core packages are fixed to a single version matching the Editor version.
