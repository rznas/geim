### Description

Structure describing device location.

### Properties

| Property | Description |
| --- | --- |
| altitude | Geographical device location altitude in meters. |
| horizontalAccuracy | Horizontal accuracy radius of the location in meters. |
| latitude | Geographical device location latitude. |
| longitude | Geographical device location longitude. |
| timestamp | Timestamp (in milliseconds from 1970) of when location data was last updated. |
| verticalAccuracy | Vertical accuracy radius of the location in meters. |
