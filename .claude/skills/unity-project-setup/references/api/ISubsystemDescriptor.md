### Description

A subsystem descriptor is metadata about a subsystem which can be inspected before loading / initializing a subsystem.

### Properties

| Property | Description |
| --- | --- |
| id | A unique string that identifies the subsystem that this Descriptor can create. |

### Public Methods

| Method | Description |
| --- | --- |
| Create | Creates an ISubsystem from this descriptor. |
