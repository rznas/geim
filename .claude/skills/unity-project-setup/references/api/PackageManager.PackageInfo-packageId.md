### Description

A read-only string property that provides the full package identifier, including the package name and version. For example, "com.unity.textmeshpro@3.0.6".

- Combines package name and version with the '@' separator.
- Used for exact package identification.
- Useful for package installation and version tracking.
- Format matches package manifest entries.
