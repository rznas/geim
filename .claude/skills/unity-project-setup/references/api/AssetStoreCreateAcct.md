# Set up an Asset Store publisher profile

To create package drafts and upload assets to them you need a Unity ID and a publisher profile. A publisher profile is connected to your Unity account. Use it to create and publish packages, and manage sales and revenue.

## Create a Unity ID

To create a Unity ID:

1. Go to [Create a Unity ID](https://id.unity.com/account/new), or open the Unity **Asset Store**.
2. Select the account icon.
3. Select **Create a Unity ID**.
4. Enter the required information.

Use a publisher profile to create, submit, and gain revenue from your packages on the Asset Store. Before you do this read the [Asset Store Provider Agreement](https://unity3d.com/legal/as_provider) for legal information, and the [Submission Guidelines](https://unity3d.com/asset-store/sell-assets/submission-guidelines) to learn about the submission process. For an example of a publisher profile page, refer to Unity’s own [Publisher’s Account page](https://assetstore.unity.com/publishers/1).

## Create a publisher profile

To create a publisher profile:

1. Open the [Asset Store Publisher Portal](https://publisher.unity.com).
2. Select the **Profile** tab.
3. Enter the required information in each field.
4. Select **View on Asset Store** to preview your publisher profile on the Asset Store.

## Set up a publisher profile

The **Profile** tab contains the following personal information associated with your Publisher Account:

- Basic info
- Contact
- Payout

### Basic Info

Use the Profile section to update your picture, name, websites, email addresses, and phone numbers for your account.

| **Property** | **Description:** |
| --- | --- |
| **Profile picture** | Your logo or brand image. This image must not be offensive or vulgar and you must have the rights or permission to use it. Don’t use Unity’s logo. |
| **Profile Name** | The name you want to associate with your publisher account. This appears on your account throughout the Asset Store, both internally and publicly.   Make sure this name doesn’t violate any trademarks or brands. |
| **Website** | Enter the address of the homepage of your website. Your website or portfolio must look professional and contain a contact page. |
| **Promo banner** | The image that appears at the top of your Asset Store page when you enter a discount period. |
| **Introduction** | A headline and paragraph where you can add technical and marketing information about your package. |
| **About** | Optional Information about yourself, for example your work experience or hobbies, that can help connect you with users on a more personal level. |

### Contact

| **Property** | **Description:** |
| --- | --- |
| **Customer Support** | A link to the support page on your website and the email address that customers can use to contact you for help with your packages. |
| **Business contact** | Your business contact information. The Asset Store team’s might use this to contact you about promotions or other business-related issues. This contact information isn’t publicly visible. |
| **Social Media** | Links to social media sites where users can read about your work, ask questions, or connect with your community. Select up to five social media sites you want to add a link to, and add the full URL in each text box. For example, you can add a link like this for a YouTube page: `https://www.youtube.com/user/<your_id>`. |

### Payout

| **Property** | **Description:** |
| --- | --- |
| **Payout preferences** | A link to your publisher account where you can define the bank account that receives sales revenue from this publisher account. For more information about Asset Store package revenue, refer to Collect revenue from Asset Store package sales. |
