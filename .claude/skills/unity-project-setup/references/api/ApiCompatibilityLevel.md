### Description

.NET API compatibility level.

### Properties

| Property | Description |
| --- | --- |
| NET_2_0 | .NET 2.0. |
| NET_2_0_Subset | .NET 2.0 Subset. |
| NET_4_6 | Deprecated, use ApiCompatibilityLevel.NET_Unity_4_8 instead. |
| NET_Web | Web profile, formerly used only by Samsung TV. |
| NET_Micro | Micro profile, used by Mono scripting backend on iOS, tvOS, and Android if stripping level is set to "Use micro mscorlib". |
| NET_Standard_2_0 | Deprecated, use ApiCompatibilityLevel.NET_Standard instead. |
| NET_Standard | Profile that targets .NET Standard 2.1. |
| NET_Unity_4_8 | Profile that targets the .NET Framework 4.8 + Unity additions. |
