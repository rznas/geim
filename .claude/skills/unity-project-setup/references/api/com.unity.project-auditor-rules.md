# Project Auditor Rules

com.unity.project-auditor-rules

## Description

Project Auditor is a static analysis tool that analyzes assets, settings, and scripts inside Unity projects. Project Auditor lives inside the Unity Editor as a module, and uses the rules in this package to provide suggestions for improvement.
