### Description

The accessibility node behaves like an image.

If this role is set on a node, the screen reader announces the node as an "image". 
 
 On Windows, if the node represents a clickable image, then it should have a role that supports AccessibilityNode.invoked, such as AccessibilityRole.Button, instead of this role.
