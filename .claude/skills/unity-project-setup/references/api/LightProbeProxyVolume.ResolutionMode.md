### Description

The resolution mode for generating a grid of interpolated Light Probes.

Additional resources: Light Probes, Light Probe Proxy Volume.

### Properties

| Property | Description |
| --- | --- |
| Automatic | The automatic mode uses a number of interpolated Light Probes per unit area, and uses the bounding volume size to compute the resolution. The final resolution value is a power of 2. |
| Custom | The custom mode allows you to specify the 3D grid resolution. |
