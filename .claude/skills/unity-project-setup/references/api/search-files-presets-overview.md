# Search files and presets

The Files and Presets search providers run implicitly as part of the Project and Asset Database search providers, but you can also run them as their own providers.

| Provider | Description |
| --- | --- |
| **Search for files** | Finds files in the `Assets` folder. |
| **Search for presets** | Finds presets in the project. |
