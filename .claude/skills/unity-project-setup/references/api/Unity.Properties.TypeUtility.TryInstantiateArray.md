### Parameters

| Parameter | Description |
| --- | --- |
| count | The count the array should have. |
| instance | When this method returns, contains the created instance, if type instantiation succeeded; otherwise, the default value for `TArray`. |

### Returns

**bool** `true` if the type was instantiated; otherwise, `false`.

### Description

Tries to create a new instance of an array with the given count.
