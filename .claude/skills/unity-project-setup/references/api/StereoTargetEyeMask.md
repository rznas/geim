### Description

Enum values for the Camera's targetEye property.

### Properties

| Property | Description |
| --- | --- |
| None | Do not render either eye to the HMD. |
| Left | Render only the Left eye to the HMD. |
| Right | Render only the right eye to the HMD. |
| Both | Render both eyes to the HMD. |
