# Search the Unity Editor interface and actions

Find and perform actions in the Unity Editor interface.

| **Provider** | **Description** |
| --- | --- |
| Search settings and preferences | Search all Unity Editor settings. |
| Search the Unity Editor main menu | Search the Unity Editor main menu and execute actions. |
| Execute Static API methods | Search the Unity API and execute actions. |
| Calculate formulas | Perform calculations that you can copy to the clipboard or save as a reusable search query. |
