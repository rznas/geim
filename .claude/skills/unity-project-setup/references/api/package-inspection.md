# Package inspection

Inspect the contents of any package in your project.

You can view information about the package in a dedicated ****Inspector**** window and access the package’s manifest file.

| **Topic** | **Description** |
| --- | --- |
| **Package Manifest window reference** | Learn about the elements in an **Inspector** window dedicated to displaying information about packages. |
| **Inspect packages** | Use the window to get information about the selected package or open the related **package manifest** file in a text editor. |
