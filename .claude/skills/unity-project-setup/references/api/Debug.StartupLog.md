### Description

Information about a message

Represents a log message consisting of a timestamp, LogType, and message.

### Properties

| Property | Description |
| --- | --- |
| logType | Type of logged message e.g. warn or error etc. |
| message | Text of logged message. |
| timestamp | Time of logged message in nanoseconds. |
