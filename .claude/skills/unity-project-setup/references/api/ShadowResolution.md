### Description

Default shadow resolution. Each decrease in quality level halves the resolution of shadows.

Additional resources: QualitySettings.shadowResolution, Light.shadowResolution, Shadow map size calculation.

### Properties

| Property | Description |
| --- | --- |
| Low | Low shadow map resolution. |
| Medium | Medium shadow map resolution. |
| High | High shadow map resolution. |
| VeryHigh | Very high shadow map resolution. |
