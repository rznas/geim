# Sculpt height with the Noise tool

**Note:** This tool is only available in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

The Noise tool is a version of the Raise and Lower Terrain tool that always includes a Noise output on the brush. Use the Noise tool to add variety to your **terrain** without creating multiple brushes. For example, you can use the Noise tool to turn a blank terrain into a more natural-looking landscape, before adding specific features with other tools.

**Tip:** Because the Noise tool is a variation on the Raise and Lower Terrain tool, it always adds height, not just details. To minimize the additional height, reduce the brush’s opacity.

## Access the tool

To use the Noise tool, go to the Terrain tile’s **Inspector** window > **Paint Terrain** tool > tool dropdown > **Sculpt** > **Noise**.

## Tool options

The Noise tool is brush-based. To learn about working with brushes, refer to Brushes.
