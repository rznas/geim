### Description

The type of the current process. It can be a Unity master instance, or a secondary instance connected to the master.

### Properties

| Property | Description |
| --- | --- |
| Main | The current process is a main instance of UnityEditor. |
| Secondary | The current process is a secondary instance of UnityEditor that is connected to a master instance. |
