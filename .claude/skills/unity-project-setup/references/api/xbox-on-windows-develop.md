# Develop Xbox games for Windows with Microsoft GDK

Use the Microsoft Game Development Kit (GDK) with Unity to develop Xbox games for Windows PC. These games can integrate Xbox services and support publishing to the Microsoft Store.

**Important**: This section covers development for Windows PC only and doesn’t cover Xbox console development. Developing for Xbox consoles requires a specific Unity license. For more information, refer to [Game development for console platforms](https://unity.com/solutions/console).

Refer to the following topics for information about how to configure your Unity project for GDK integration and implement features, such as in-app purchases.

| **Topic** | **Description** |
| --- | --- |
| **Set up a Unity project for Xbox games on Windows** | Configure a Unity project to use Microsoft GDK for Windows development. |
| **In-app purchases for Xbox games on Windows** | Set up in-app purchases for Xbox games that run on Windows PC. |
