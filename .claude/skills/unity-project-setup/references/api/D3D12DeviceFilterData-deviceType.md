### Description

Defines the type of the GPU as D3D12GraphicsDeviceType.

Select `Discrete`, `Integrated` or `DontCare` as the type of GPU to apply the filter to. For example, `Integrated`.

 **Note**: Seleting `DontCare` means that the device type does not affect the filter.
