### Description

Methods to compensate for aspect ratio discrepancies between the source resolution and the wanted encoding size.

### Properties

| Property | Description |
| --- | --- |
| NoScaling | Perform no operation. |
| Stretch | Stretch the source to fill the target resolution without preserving the aspect ratio. |
