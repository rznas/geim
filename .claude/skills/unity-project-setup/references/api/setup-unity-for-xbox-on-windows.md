# Set up a Unity project for Xbox games on Windows

Set up your Unity project to use the Microsoft Game Development Kit (GDK) to develop Xbox games for Windows PC.

## Prerequisites

Before you begin, review Unity’s Windows requirements and compatibility information. Ensure that you have the following:

- [Microsoft Game Development Kit (GDK)](https://learn.microsoft.com/en-us/gaming/gdk/?view=gdk-2604)
- [A Microsoft Partner Center account](https://learn.microsoft.com/en-us/partner-center/)
- Enrollment in the [ID@Xbox program](https://developer.microsoft.com/en-US/games/publish)

## Microsoft GDK for Unity

For setup instructions, refer to the Microsoft documentation on [PC end-to-end guide overview](https://learn.microsoft.com/en-us/gaming/gdk/docs/gdk-dev/pc-dev/tutorials/pc-e2e-guide/e2e-overview?view=gdk-2604). This documentation describes the workflow for integrating the GDK with your game, including GDK initialization, player sign-in, achievements, cloud saves, and publishing to the Microsoft Store.

**Note**: It’s recommended to use Unity’s Microsoft GDK Tools and API packages that provide tools and workflow support for the development of Unity projects using the Microsoft Game Development Kit (GDK).
