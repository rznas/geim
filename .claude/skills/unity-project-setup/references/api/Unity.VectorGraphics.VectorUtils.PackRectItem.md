### Description

Struct to hold a texture atlas location.

### Properties

| Property | Description |
| --- | --- |
| Fill | The fill associated with this entry, may be null. |
| Position | The position of the entry inside the atlas. |
| Rotated | True if the entry is rotated by 90 degrees. |
| Size | The size of the entry inside the atlas. |
