# Default Scene view overlay reference

From the Overlays menu, you can control which overlays are in the Scene view and manage your overlay configurations. To display the Overlays menu in the **Scene** view, press the **`** key. To restore the default overlay configuration, select the **Default** overlay configuration from the **Overlays menu**.

By default, the following overlays display in the **Scene view**.

| **Overlay** | **Description** |
| --- | --- |
| **Tools** | Use the Tools overlay to select a tool context, use Transform tools, and, if they’re available, component tools. |
| **Tool Settings** | Use the Tool Settings overlay to display available settings for the tool you have selected. For example, if you select a Transform tool, use the Tool Settings overlay to select the location of any Transform tool Gizmo and the handles used to manipulate the Gizmo. |
| **Grid and Snap** | Control how **GameObjects** snap to the grid. To learn more, refer to Grid and Snap overlay. |
| **Draw Modes** | Control the appearance of meshes in the Scene view. To learn more, refer to Draw Modes overlay reference. |
| **View Options** | Use the View Options overlay to select view options, control lighting, and control audio for the Scene view. These controls only affect the Scene view and don’t affect the built game. |
| **Orientation** | Use the Orientation overlay to view the Scene **camera**’s current orientation and to change its viewing angle and projection mode. |
| **Overlay Menu** | Show or hide each overlay in the Scene view. To learn more, refer to Overlays menu reference. |
