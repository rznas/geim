### Description

Represents an asynchronous request to reset the project packages to the current Editor default configuration.

The PackageManager Client class returns a ResetToEditorDefaultsRequest instance when you call the Client. ResetToEditorDefaultsRequest method to reset the packages in your project to the current Editor defaults. Use this object to determine when the request is complete.
