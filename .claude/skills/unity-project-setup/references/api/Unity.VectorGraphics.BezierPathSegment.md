### Description

A bezier path segment.

Like BezierSegment but implies connectivity of segments, where `segments[0].P3` is actually `segments[1].P0`.

### Properties

| Property | Description |
| --- | --- |
| P0 | Origin point of the segment. |
| P1 | First control point of the segment. |
| P2 | Second control point of the segment. |
