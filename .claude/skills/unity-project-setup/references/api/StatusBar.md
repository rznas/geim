# Status bar

The status bar provides notifications about various Unity processes, and quick access to related tools and settings. You cannot move or rearrange the status bar.

The status bar displays the following:

1. The most recent message logged to the Console window. Click the message to open the Console window.
2. A global progress bar for various asynchronous tasks (for example, shader compilation, lightmap baking, and occlusion culling). Click the progress bar to open the Background Tasks window, which displays progress for individual tasks and subtasks.
3. The current code optimization mode. Click the code optimization icon to switch between debug mode and release mode.
4. The cache server status. Click the icon to get additional information about the cache server, and re-establish a lost connection.
5. An activity indicator (spinner) that shows when Unity compiles C# **scripts** or runs asynchronous tasks.
