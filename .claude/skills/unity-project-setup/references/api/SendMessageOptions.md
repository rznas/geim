### Description

Options for how to send a message.

This is used by SendMessage & BroadcastMessage in GameObject & Component.

### Properties

| Property | Description |
| --- | --- |
| RequireReceiver | A receiver is required for SendMessage. |
| DontRequireReceiver | No receiver is required for SendMessage. |
