### Description

Updates the Light Probe Proxy Volume every frame.

This may be resource-intensive. The performance impact depends on the resolution of the interpolated Light Probe grid. The Light Probe interpolation is multi-threaded.

Additional resources: Light Probe Proxy Volume.
