### Description

A chain of bezier paths, optionnally closed.

For more information, refer to work with vector graphics.

### Properties

| Property | Description |
| --- | --- |
| Closed | A boolean indicating if the contour should be closed. |
| Segments | An array of every path segments on the contour. |
